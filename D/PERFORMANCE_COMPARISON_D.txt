3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ D
================================================================================

Совет 1: Используйте LDC вместо DMD
------------------------------------
// МЕДЛЕННО (DMD):
dmd -O program.d

// БЫСТРО (LDC):
ldc2 -O3 program.d

// ⚡ LDC быстрее в runtime!

Совет 2: Отключите GC для performance-critical кода
---------------------------------------------------
// МЕДЛЕННО (с GC):
int[] arr = new int[1000000];

// БЫСТРО (без GC):
@nogc {
    int* arr = new int[1000000];
    delete arr;
}

// ⚡ @nogc предотвращает GC паузы!

Совет 3: Используйте CTFE для compile-time вычислений
-----------------------------------------------------
// МЕДЛЕННО (runtime):
int result = factorial(10);

// БЫСТРО (compile-time):
enum result = factorial(10);

// ⚡ CTFE выполняется на компиляции!

Совет 4: Используйте parallel algorithms
----------------------------------------
// МЕДЛЕННО (последовательно):
import std.algorithm;
auto result = data.map!(x => x * 2);

// БЫСТРО (параллельно):
import std.parallelism;
auto result = data.parallel.map!(x => x * 2);

// ⚡ Parallel ускоряет на multi-core!

Совет 5: Используйте scope для RAII
-----------------------------------
// МЕДЛЕННО (manual cleanup):
auto resource = acquire();
// ...
resource.release();

// БЫСТРО (RAII):
scope(exit) resource.release();

// ⚡ Scope гарантирует cleanup!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.d:
---------------------
import std.stdio;
import std.datetime.stopwatch;

int sumEnum(int n) {
    int sum = 0;
    for (int i = 1; i <= n; i++) {
        sum += i;
    }
    return sum;
}

void main() {
    auto sw = StopWatch(AutoStart.yes);
    auto result = sumEnum(1_000_000_000);
    sw.stop();
    
    writeln("Sum: ", result);
    writeln("Time: ", sw.peek.total!"msecs", " ms");
}

Компиляция и запуск:
--------------------
ldc2 -O3 -release benchmark.d
./benchmark

Сравнение с C++:
----------------
# D будет сопоставим с C++ по производительности!
