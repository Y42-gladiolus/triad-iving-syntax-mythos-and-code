3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ DHALL
================================================================================

Совет 1: Кэшируйте импорты
--------------------------
# МЕДЛЕННО (каждый раз download):
let config = https://example.com/config.dhall

# БЫСТРО (кэш):
dhall cache
let config = https://example.com/config.dhall

# ⚡ Кэш ускоряет повторные компиляции!

Совет 2: Используйте локальные файлы
------------------------------------
# МЕДЛЕННО (network):
let config = https://example.com/config.dhall

# БЫСТРО (локально):
let config = ./config.dhall

# ⚡ Локальные файлы быстрее!

Совет 3: Нормализуйте заранее
-----------------------------
# МЕДЛЕННО (нормализация каждый раз):
dhall-to-json < config.dhall

# БЫСТРО (пре-нормализация):
dhall normalize config.dhall > config-normalized.dhall
dhall-to-json < config-normalized.dhall

# ⚡ Пре-нормализация ускоряет CI/CD!

Совет 4: Используйте бинарные релизы
------------------------------------
# МЕДЛЕННО (компиляция из Haskell):
cabal install dhall

# БЫСТРО (бинарный релиз):
wget https://github.com/dhall-lang/dhall-haskell/releases/...

# ⚡ Бинарные релизы быстрее!

Совет 5: Минимизируйте импорты
------------------------------
# МЕДЛЕННО (много импортов):
let Prelude = ...
let List = ...
let Optional = ...

# БЫСТРО (минимум импортов):
let Prelude = ...  -- Только что нужно

# ⚡ Меньше импортов = быстрее компиляция!

8. ИНТЕГРАЦИЯ DHALL С ДРУГИМИ ЯЗЫКАМИ
================================================================================

Dhall + Kubernetes:
-------------------
dhall-to-json < deployment.dhall > deployment.json
kubectl apply -f deployment.json

Dhall + Terraform:
------------------
dhall-to-json < terraform.dhall > terraform.json
terraform apply

Dhall + Nix:
------------
let nix = import ./package.dhall
in  nix.mkDerivation { ... }

Dhall + Docker:
---------------
dhall-to-json < docker.dhall > docker-compose.json
docker-compose up

Dhall + CI/CD:
--------------
# .github/workflows/ci.yml
- name: Validate Dhall
  run: dhall type < config.dhall

- name: Compile to JSON
  run: dhall-to-json < config.dhall > config.json
