1. УСТАНОВКА JULIA
================================================================================

Linux (Ubuntu/Debian):
----------------------
# Официальный способ (рекомендуется)
wget https://julialang-s3.julialang.org/bin/linux/x64/1.10/julia-1.10.0-linux-x86_64.tar.gz
tar zxvf julia-1.10.0-linux-x86_64.tar.gz
sudo mv julia-1.10.0 /opt/julia
sudo ln -s /opt/julia/bin/julia /usr/local/bin/julia

# Проверка
julia --version

# Альтернатива: через пакетный менеджер (может быть старая версия)
sudo apt install julia

macOS (Homebrew):
-----------------
brew install julia

# Проверка
julia --version

Windows:
--------
1. Скачать с https://julialang.org/downloads/
2. Запустить установщик .exe
3. Добавить в PATH при установке
4. Проверка в PowerShell: julia --version

Docker (универсально):
----------------------
docker run -it --rm julia julia

4. УПРАВЛЕНИЕ ПАКЕТАМИ (PKG)
================================================================================

# Вход в Pkg режим (в REPL):
]

# Установка пакета:
pkg> add PackageName

# Выход из Pkg режима:
pkg> backspace

# Или прямо из Julia:
using Pkg
Pkg.add("PackageName")

# Основные команды:
] add PackageName           # Установить пакет
] update                    # Обновить все пакеты
] status                    # Показать установленные пакеты
] rm PackageName            # Удалить пакет
] test PackageName          # Запустить тесты пакета
] build PackageName         # Пересобрать пакет

# Установка нескольких пакетов:
] add Distributions JuMP Optim

# Установка конкретной версии:
] add PackageName@1.2.3

# Установка из GitHub:
] add https://github.com/user/Repo.jl

# Активация проекта:
] activate .                # Активировать текущую директорию
] activate myproject        # Активировать проект по имени

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Время первой компиляции (TTFP - Time To First Plot)
   Первый запуск функции может быть медленным (1-10 сек)
   
   Решение: Использовать PackageCompiler.jl для создания системных образов
   ] add PackageCompiler
   using PackageCompiler
   create_sysimage(:YourPackage)

⚠️ Проблема 2: Глобальные переменные медленные
   Глобальные переменные в Julia медленные!
   
   Решение: Обернуть код в функции
   # ПЛОХО:
   x = 5
   for i in 1:1000
       x += i
   end
   
   # ХОРОШО:
   function my_function()
       x = 5
       for i in 1:1000
           x += i
       end
       return x
   end

⚠️ Проблема 3: Нестабильные типы
   Julia компилирует код для конкретных типов
   
   Решение: Избегать типов Any, использовать конкретные типы
   # ПЛОХО:
   x::Any = 5
   
   # ХОРОШО:
   x::Int = 5

⚠️ Проблема 4: Много памяти при компиляции
   Julia кэширует скомпилированный код
   
   Решение: Очистить кэш при необходимости
   rm -rf ~/.julia/compiled

⚠️ Проблема 5: Конфликты версий пакетов
   Разные пакеты могут требовать разные версии зависимостей
   
   Решение: Использовать изолированные проекты (environments)
   ] activate myproject
   ] instantiate

⚠️ Проблема 6: Кодировка (Windows, кириллица)
   Решение: Сохранять файлы в UTF-8

# Добавить в ~/.bashrc, ~/.zshrc, ~/.mkshrc

echo "alias jl='julia'" >> ~/.bashrc
echo "alias jl='julia'" >> ~/.zshrc

# Для запуска скриптов
echo "alias jls='julia -e'" >> ~/.bashrc
echo "alias jls='julia -e'" >> ~/.zshrc

# Применить изменения
source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true

# Теперь:
# jl          → запуск Julia REPL
# jls 'code'  → выполнить код
