3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ DART
================================================================================

Совет 1: Используйте AOT для production
---------------------------------------
// МЕДЛЕННО (JIT - разработка):
dart run bin/my_app.dart

// БЫСТРО (AOT - production):
dart compile exe bin/my_app.dart
./bin/my_app.exe

// ⚡ Ускорение: 2-5x для startup time!

Совет 2: Используйте const для immutable данных
-----------------------------------------------
// МЕДЛЕННО (runtime создание):
var list = [1, 2, 3];

// БЫСТРО (compile-time константа):
const list = [1, 2, 3];

// ⚡ Ускорение: меньше аллокаций памяти!

Совет 3: Используйте async/await правильно
------------------------------------------
// МЕДЛЕННО (последовательно):
var result1 = await fetch1();
var result2 = await fetch2();
var result3 = await fetch3();

// БЫСТРО (параллельно):
var results = await Future.wait([
  fetch1(),
  fetch2(),
  fetch3(),
]);

// ⚡ Ускорение: 3x для независимых запросов!

Совет 4: Используйте Stream для больших данных
----------------------------------------------
// МЕДЛЕННО (загрузить всё в память):
var data = await loadAllData();
process(data);

// БЫСТРО (поток):
await loadData().forEach((item) => process(item));

// ⚡ Ускорение: меньше памяти!

Совет 5: Используйте isolate для CPU-bound
------------------------------------------
// МЕДЛЕННО (блокирует UI):
var result = heavyComputation();

// БЫСТРО (отдельный isolate):
var isolate = await Isolate.spawn(heavyComputation, data);

// ⚡ Ускорение: UI не блокируется!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.dart:
------------------------
import 'dart:io';

int sumEnum(int n) {
  var sum = 0;
  for (var i = 1; i <= n; i++) {
    sum += i;
  }
  return sum;
}

void main() async {
  var stopwatch = Stopwatch()..start();
  var result = sumEnum(1_000_000);
  stopwatch.stop();
  
  print('Sum: $result');
  print('Time: ${stopwatch.elapsedMilliseconds} ms');
}

Запуск:
-------
dart run benchmark.dart
dart compile exe benchmark.dart && ./benchmark.exe
