3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ WREN
================================================================================

Совет 1: Используйте fibers для I/O
-----------------------------------
# МЕДЛЕННО (blocking):
var data = readFile("file.txt")

# БЫСТРО (fiber):
var fiber = Fiber.new {
  var data = yield readFile("file.txt")
}

# ⚡ Fibers для cooperative multitasking!

Совет 2: Используйте lists вместо maps для small collections
------------------------------------------------------------
# МЕДЛЕННО (map):
var map = {"a": 1, "b": 2, "c": 3}
var x = map["a"]

# БЫСТРО (list):
var list = [1, 2, 3]
var x = list[0]

# ⚡ Lists быстрее для small collections!

Совет 3: Избегайте лишних object creation
-----------------------------------------
# МЕДЛЕННО (creation в loop):
for (i in 0..1000) {
  var obj = Object.new()
}

# БЫСТРО (reuse):
var obj = Object.new()
for (i in 0..1000) {
  // Reuse obj
}

# ⚡ Object creation имеет overhead!

Совет 4: Используйте static methods для utility functions
---------------------------------------------------------
# МЕДЛЕННО (instance methods):
var math = Math.new()
var result = math.add(5, 3)

# БЫСТРО (static methods):
var result = Math.add(5, 3)

# ⚡ Static methods быстрее!

Совет 5: Минимизируйте Fiber.yield вызовы
----------------------------------------
# МЕДЛЕННО (много yield):
for (i in 0..1000) {
  Fiber.yield(i)
}

# БЫСТРО (batch yield):
var batch = []
for (i in 0..1000) {
  batch.add(i)
  if (batch.count == 100) {
    Fiber.yield(batch)
    batch = []
  }
}

# ⚡ Меньше yield = быстрее!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.wren:
------------------------
class Main {
  static main() {
    var start = System.clock()
    var sum = 0
    for (i in 1..1000000000) {
      sum = sum + i
    }
    var end = System.clock()
    System.print("Sum: %(sum)")
    System.print("Time: %((end - start) * 1000) ms")
  }
}

Запуск:
-------
wren benchmark.wren
