3. БЫСТРЫЙ СТАРТ
================================================================================

# Wren REPL
wren

# В REPL:
> System.print("Hello, Wren!")
> 2 + 2
> Fiber.yield()

# Создание проекта
mkdir my_project && cd my_project

# Структура проекта:
my_project/
├── main.wren      # Главный файл
├── lib/           # Библиотеки
└── README.md      # Документация

# Простая программа (main.wren)
class Main {
  static main() {
    System.print("Hello, Wren!")
  }
}

# Запуск
wren main.wren

# Запуск с кодом из командной строки
wren -e "System.print(\"Hello\")"

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Меньше библиотек
   Экосистема меньше чем Lua/Python
   
   Решение:
   Использовать C API для интеграции
   Писать свои библиотеки

⚠️ Проблема 2: Документация
   Документация ещё развивается
   
   Решение:
   Читать wren.io
   Присоединиться к Discord

⚠️ Проблема 3: IDE поддержка
   Ограниченная поддержка
   
   Решение:
   Использовать VS Code + Wren extension
   Или текстовый редактор + CLI

⚠️ Проблема 4: Нет package manager
   Manual dependency management
   
   Решение:
   Clone repositories manually
   Follow project updates

⚠️ Проблема 5: Dynamic typing
   Runtime type errors возможны
   
   Решение:
   Использовать testing
   Write defensive code

⚠️ Проблема 6: No threads
   Только fibers (cooperative)
   
   Решение:
   Использовать fibers для concurrency
   Или multiple VMs для parallelism

source ~/.zshrc
