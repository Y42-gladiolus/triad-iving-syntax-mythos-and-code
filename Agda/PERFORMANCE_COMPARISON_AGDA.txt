3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ AGDA
================================================================================

Совет 1: Используйте --fast флаг
---------------------------------
# МЕДЛЕННО (полная проверка):
agda file.agda

# БЫСТРО (для разработки):
agda --fast file.agda

-- ⚡ Ускорение: 2-5x для разработки!

Совет 2: Избегайте доказательств в runtime коде
-----------------------------------------------
-- МЕДЛЕННО (доказательство в runtime):
process : (xs : List A) → (h : length xs > 0) → A
process xs h = head xs

-- БЫСТРО (доказательство в типе):
process : (xs : Vec A (succ n)) → A
process xs = head xs

-- ⚡ Доказательство проверяется на compile-time!

Совет 3: Используйте with для сложных случаев
---------------------------------------------
-- МЕДЛЕННО (вложенные match):
f x = case x of
  zero → ...
  succ n → case n of
    zero → ...
    succ m → ...

-- БЫСТРО (with):
f zero = ...
f (succ n) with n
... | zero = ...
... | succ m = ...

-- ⚡ Чище и быстрее для компилятора!

Совет 4: Кэшируйте .agdai файлы
-------------------------------
# Первый build (медленно):
agda file.agda

# Последующие (быстро, использует .agdai):
agda file.agda

-- Кэш хранится в .agdai файлах

Совет 5: Используйте модульную организацию
------------------------------------------
-- ПЛОХО (один большой файл):
module Everything where
  -- 1000 строк кода

-- ХОРОШО (модули):
module Part1 where ...
module Part2 where ...
module Main where
  open Part1
  open Part2

-- ⚡ Быстрее компиляция и проверка!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте Benchmark.agda:
------------------------
module Benchmark where

open import Data.Nat
open import Data.Nat.Properties

sumTo : ℕ → ℕ
sumTo zero = zero
sumTo (succ n) = n + sumTo n

theorem : ∀ n → 2 * sumTo n ≡ n * (n + 1)
theorem zero = refl
theorem (succ n) = 
  begin
    2 * (n + sumTo n)
  ≡⟨ *-distribˡ-+ 2 n (sumTo n) ⟩
    2 * n + 2 * sumTo n
  ≡⟨ cong (2 * n +_) (theorem n) ⟩
    2 * n + n * (n + 1)
  ≡⟨ ? ⟩
    (succ n) * (succ n + 1)
  ∎

Компиляция и проверка:
----------------------
agda Benchmark.agda

-- В Emacs: C-c C-l для проверки
