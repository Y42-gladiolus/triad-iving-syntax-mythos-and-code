Linux (Ubuntu/Debian):
----------------------
# Шаг 1: Установить GHC и cabal
sudo apt update
sudo apt install ghc cabal-install happy alex

# Шаг 2: Обновить cabal
cabal update

# Шаг 3: Установить Agda
cabal install Agda

# Шаг 4: Установить agda-mode (для Emacs)
~/.cabal/bin/agda-mode setup

# Шаг 5: Добавить в PATH
export PATH="$HOME/.cabal/bin:$PATH"

# Проверка
agda --version
agda-mode --version

macOS (Homebrew):
-----------------
brew install agda
agda-mode setup

# Проверка
agda --version

Установка Emacs + agda-mode:
----------------------------
# После установки Agda:
~/.cabal/bin/agda-mode setup

# Перезапустить Emacs
# Проверить: M-x agda-mode-version

Основные команды agda-mode:
---------------------------
C-c C-l  - Загрузить файл (type check)
C-c C-c  - Проверить тип (Check)
C-c C-d  - Показать тип выражения (Describe)
C-c C-n  - Нормализовать терм (Normalize)
C-c C-r  - Рефинировать hole (Refine)
C-c C-,  - Показать контекст и цель (Goal type)
C-c C-a  - Автоматическое доказательство (Auto)
C-c C-t  - Показать тип цели (Type check goal)
