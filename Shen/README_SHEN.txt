Linux (Ubuntu/Debian):
----------------------
# Вариант 1: Через пакетный менеджер (Shen/Scheme)
sudo apt update
sudo apt install shen-scheme

# Вариант 2: Из исходников (рекомендуется!)
git clone https://github.com/shen-lang/shen.git
cd shen
make
sudo make install

# Вариант 3: Через Racket
raco pkg install shen

# Проверка
shen --version
# или
shen-scheme

macOS (Homebrew):
-----------------

git clone https://github.com/shen-lang/shen.git

3. БЫСТРЫЙ СТАРТ
================================================================================

# Shen REPL
shen

# В REPL:
(1+) 2 2
--> 4 : number
(1+) (quit)  # Выход

# Запуск скрипта
shen script.shen

# Запуск с кодом из командной строки
shen -e "(+ 2 2)"

# Компиляция в Scheme
shen --compile script.shen

# Создание проекта
mkdir my_project && cd my_project

# Структура проекта:
my_project/
├── main.shen      # Главный файл
├── lib/           # Библиотеки
│   └── utils.shen
├── tests/         # Тесты

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Требует Scheme или Common Lisp
   Shen компилируется в другие Lisps
   
   Решение:
   Установить Chez Scheme или Common Lisp
   sudo apt install chez-scheme

⚠️ Проблема 2: Статическая типизация непривычна
   Для Lisp-еров это ново
   
   Решение:
   Читать документацию по типам
   Практиковаться ежедневно

⚠️ Проблема 3: Мало документации
   Меньше чем Common Lisp
   
   Решение:
   Читать "Shen: A Programming Language"
   Изучать примеры на GitHub

⚠️ Проблема 4: Pattern matching сложный
   Требует практики
   
   Решение:
   Изучать примеры паттернов
   Практиковаться с REPL

⚠️ Проблема 5: Компиляция медленная
   Первый запуск может занять время
   
   Решение:
   Это нормально - компиляция в Scheme
   Последующие запуски быстрые

⚠️ Проблема 6: Разные реализации
   Shen/Scheme, Shen/CL, Shen/JS
   
   Решение:
   Использовать Shen/Scheme для совместимости
   Проверять совместимость

echo "alias shc='shen --compile'" >> ~/.zshrc

echo "alias she='shen -e'" >> ~/.bashrc
echo "alias she='shen -e'" >> ~/.zshrc

# Применить изменения
source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true

shen test.shen
rm test.shen

# Тест 4: Проверка типов
echo '(type 2 number)' | shen
