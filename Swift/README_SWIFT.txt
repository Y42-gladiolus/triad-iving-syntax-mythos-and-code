1. УСТАНОВКА
================================================================================

macOS (через Xcode - рекомендуемый способ):
-------------------------------------------
1. App Store → Xcode → Install (5-10 GB)
2. После установки:
   xcode-select --install
3. Проверка:
   swift --version

Linux (Ubuntu/Debian):
----------------------
# Официальный способ
sudo apt update
sudo apt install swift-lang

# Или скачать с swift.org
# https://swift.org/download/

# Проверка
swift --version

Docker (универсально):
----------------------
docker run -it --rm swift:5.9 swift

3. БЫСТРЫЙ СТАРТ
================================================================================

# Создание проекта
mkdir MyProject && cd MyProject
swift package init --type executable

# Структура:
MyProject/
├── Package.swift
├── Sources/
│   └── MyProject/
│       └── main.swift
└── Tests/

# Запуск
swift run

# Сборка
swift build

# Тесты
swift test

4. ОДНОФАЗНЫЙ ЗАПУСК (как скрипт)
================================================================================

# Файл: hello.swift
print("Hello, World!")

# Запуск без компиляции
swift hello.swift

# Или с shebang
#!/usr/bin/env swift
print("Hello from script!")

# chmod +x hello.swift
# ./hello.swift

5. ПОДВОДНЫЕ КАМНИ
================================================================================

⚠️ Версии Swift:
   - Swift 5.x стабилен
   - Swift 6.0 вводит новую модель конкурентности
   - Проверьте совместимость библиотек

⚠️ Linux vs macOS:
   - Некоторые API доступны только на macOS (UIKit, AppKit)
   - Foundation работает везде
   - Тестируйте на целевой платформе

⚠️ Скорость компиляции:
   - Первый запуск медленный
   - Используйте swift build для инкрементальной сборки

⚠️ Память:
   - ARC (автоматическое управление памятью)
   - Остерегайтесь циклических ссылок (weak, unowned)
