3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ CLOJURESCRIPT
================================================================================

Совет 1: Используйте advanced compilation
-----------------------------------------
; МЕДЛЕННО (none):
:optimizations :none

; БЫСТРО (advanced):
:optimizations :advanced

; ⚡ Ускорение: 5-10x для runtime!

Совет 2: Используйте transducers
--------------------------------
; МЕДЛЕННО (chain):
(->> data
     (filter pred)
     (map transform)
     (reduce + 0))

; БЫСТРО (transducer):
(transduce (comp (filter pred) (map transform)) + 0 data)

; ⚡ Transducers эффективнее!

Совет 3: Избегайте cons в циклах
--------------------------------
; МЕДЛЕННО (cons):
(loop [lst '() i 0]
  (if (>= i 1000)
    lst
    (recur (cons i lst) (inc i))))

; БЫСТРО (vector):
(loop [v [] i 0]
  (if (>= i 1000)
    v
    (recur (conj v i) (inc i))))

; ⚡ Vectors быстрее для накопления!

Совет 4: Используйте memoize
----------------------------
; МЕДЛЕННО (без кэша):
(defn factorial [n]
  (if (<= n 1) 1 (* n (factorial (dec n)))))

; БЫСТРО (с кэшем):
(def factorial (memoize
  (fn [n]
    (if (<= n 1) 1 (* n (factorial (dec n)))))))

; ⚡ Memoize кэширует результаты!

Совет 5: Используйте core.async для I/O
---------------------------------------
; МЕДЛЕННО (callbacks):
(js/fetch url
  #(-> %
       (.json)
       (.then success-handler)
       (.catch error-handler)))

; БЫСТРО (core.async):
(go
  (let [response (<! (js->clj (js/fetch url)))
        data (<! (.json response))]
    (success-handler data)))

; ⚡ core.async чище код!

--------------------------------------

(js/console.log "Hello")
(js/fetch url)

; Использование NPM пакетов
(require '["lodash" :as _])
(_.debounce fn 300)

ClojureScript + Python (через HTTP):
------------------------------------
(go
  (let [response (<! (js/fetch "/api/python"))
        data (<! (.json response))]
    (println data)))

ClojureScript + MiniZinc (через HTTP):
--------------------------------------
(defn run-minizinc [model data]
  (js/fetch "/api/minizinc"
    #js {:method "POST"
         :body (js/JSON.stringify #js {:model model :data data})}))

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.cljs:
------------------------
(ns benchmark)

(defn sum-enum [n]
  (reduce + 0 (range n)))

(defn ^:export run []
  (let [start (.now js/Date)
        result (sum-enum 100000000)
        end (.now js/Date)]
    (js/console.log "Time:" (- end start) "ms")))

Компиляция и запуск:
--------------------
lein cljsbuild once
node benchmark.js

Сравнение с JavaScript:
-----------------------
# ClojureScript будет в 2-3 раза медленнее
# Но с лучшей выразительностью!
