3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ SMALLTALK
================================================================================

Совет 1: Используйте коллекции правильно
----------------------------------------
" МЕДЛЕННО (nested loops): "
1 to: 1000 do: [ :i |
    1 to: 1000 do: [ :j |
        sum := sum + (i * j).
    ].
].

" БЫСТРО (collection methods): "
sum := (1 to: 1000) inject: 0 into: [ :sum :i |
    sum + ((1 to: 1000) inject: 0 into: [ :sum :j | sum + (i * j) ]).
].

" ⚡ Collection methods оптимизированы! "

Совет 2: Избегайте unnecessary object creation
----------------------------------------------
" МЕДЛЕННО (создание объектов): "
1 to: 1000000 do: [ :i |
    str := i asString.
].

" БЫСТРО (reuse): "
str := String new.
1 to: 1000000 do: [ :i |
    str := i printOn: str.
].

" ⚡ Reuse объектов быстрее! "

Совет 3: Используйте primitive methods
--------------------------------------
" МЕДЛЕННО (Smalltalk code): "
sum := 0.
1 to: 1000000 do: [ :i |
    sum := sum + i.
].

" БЫСТРО (primitive): "
sum := (1 to: 1000000) inject: 0 into: [ :sum :each | sum + each ].

" ⚡ Primitive methods быстрее! "

Совет 4: Image optimization
---------------------------
" МЕДЛЕННО (большой image): "
" Накопление объектов в image "

" БЫСТРО (очистка): "
Smalltalk garbageCollect.
Smalltalk saveSession.

" ⚡ Регулярная очистка image! "

Совет 5: Use Pharo (modern implementation)
------------------------------------------
