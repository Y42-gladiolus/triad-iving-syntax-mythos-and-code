3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ EXECLINE
================================================================================

Совет 1: Используйте -S strict mode
-----------------------------------
# МЕДЛЕННО (без проверки):
#!/command/execlineb

# БЫСТРО (с проверкой):
#!/command/execlineb -S

# ⚡ Strict mode ловит ошибки раньше!

Совет 2: Импортируйте PATH
--------------------------
# МЕДЛЕННО (поиск команд):
command

# БЫСТРО (с PATH):
importas -u PATH PATH
command

# ⚡ PATH ускоряет поиск команд!

Совет 3: Используйте posix-chronic
----------------------------------
# МЕДЛЕННО (с остановкой):
command-that-might-fail
echo "Always runs"  # Не выполнится при ошибке!

# БЫСТРО (без остановки):
posix-chronic
command-that-might-fail
echo "Always runs"  # Выполнится всегда!

# ⚡ posix-chronic предотвращает остановки!

Совет 4: Используйте foreground для ожидания
--------------------------------------------
# МЕДЛЕННО (без ожидания):
background {
  long-command
}
exit 0  # Процесс убьётся!

# БЫСТРО (с ожиданием):
foreground {
  long-command
}
exit 0  # Дождётся завершения!

# ⚡ foreground обеспечивает правильное завершение!

Совет 5: Минимизируйте команды
------------------------------
# МЕДЛЕННО (много процессов):
echo "line1"
echo "line2"
echo "line3"

# БЫСТРО (одна команда):
echo "line1
line2
line3"

# ⚡ Меньше процессов = быстрее!

8. ИНТЕГРАЦИЯ EXECLINE С ДРУГИМИ ЯЗЫКАМИ
================================================================================

execline + bash (через exec):
-----------------------------
#!/command/execlineb -S
importas -u PATH PATH
exec /bin/bash script.sh

execline + Python (через exec):
-------------------------------
#!/command/execlineb -S
importas -u PATH PATH
exec python3 script.py

execline + s6:
--------------
# /etc/s6/sv/service/run
#!/command/execlineb -S
importas -u PATH PATH
exec ./service-binary

execline + Docker:
------------------
# Dockerfile
FROM alpine:latest
RUN apk add execline
ENTRYPOINT ["/command/execlineb", "-S"]
CMD ["./run"]

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.ex:
----------------------
#!/command/execlineb -S
importas -u PATH PATH
posix-chronic
foreground {
  echo "Start"
  sleep 1
  echo "End"
}

Запуск:
-------
chmod +x benchmark.ex
time ./benchmark.ex

Сравнение с bash:
-----------------
# execline будет быстрее на startup
# Но bash гибче для сложной логики
