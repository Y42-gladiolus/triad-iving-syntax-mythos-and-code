1. УСТАНОВКА FORTH (GFORTH)
================================================================================

┌──────────────────────────────────────────────────────────────────────────────┐
│                    FORTH = STACK-BASED + МИНИМАЛИЗМ                          │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  Forth используется:                                                         │
│  • Embedded системы (микроконтроллеры)                                      │
│  • Bootloaders (загрузчики ОС)                                              │
│  • Космические приложения (NASA)                                            │
│  • ROM BIOS                                                                 │
│  • Калькуляторы (HP)                                                        │
│                                                                              │
│  • Интерпретатор ~8 KB                                                      │
│  • Stack-based (постфиксная нотация)                                        │
│  • 50+ лет эволюции (1970-2024)                                             │
│  • Нет типов (в классическом Forth)                                         │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

Linux (Ubuntu/Debian):
----------------------
# Вариант 1: Через apt (Gforth)
sudo apt update
sudo apt install gforth gforth-doc

# Вариант 2: Из исходников
wget https://ftp.gnu.org/gnu/gforth/gforth-0.7.3.tar.gz
tar xzf gforth-0.7.3.tar.gz
cd gforth-0.7.3
./configure
make
sudo make install

# Проверка
gforth --version

macOS (Homebrew):
-----------------
brew install gforth

# Проверка
gforth --version

Windows:
--------
# Скачать с https://www.gnu.org/software/gforth/
# Установщик .exe
# Добавить в PATH: C:\Program Files\gforth

# Проверка в PowerShell:
gforth --version

Docker (универсально):
----------------------
docker run -it --rm joshco/gforth gforth --version

gforth

ok> 2 2 + .

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Stack overflow
   Слишком глубокая рекурсия
   
   Решение:
   Увеличить размер стека
   gforth --stack-size=1000000

⚠️ Проблема 2: Постфиксная нотация непривычна
   2 2 + вместо 2 + 2
   
   Решение:
   Практиковаться ежедневно
   Думать в терминах стека

⚠️ Проблема 3: Нет типов (классический Forth)
   Ошибки только runtime
   
   Решение:
   Использовать Forth 200x с типами
   Или тщательное тестирование

⚠️ Проблема 4: Little documentation
   Мало современной документации
   
   Решение:
   Читать "Starting Forth"
   Читать ANS Forth стандарт

⚠️ Проблема 5: Разные диалекты
   Gforth, SwiftForth, Retro, etc.
   
   Решение:
   Использовать ANS Forth для совместимости
   Или Gforth (наиболее популярный)

⚠️ Проблема 6: Отладка сложная
   Stack-based отладка непривычна
   
   Решение:
   Использовать .s для просмотра стека
   Использовать see для дизассемблирования

echo ': hello ." Hello, Forth!" ; hello bye' > test.fs
gforth test.fs

echo '1 2 3 .s bye' | gforth
