3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ PICOLISP
================================================================================

Совет 1: Используйте coroutines для I/O
---------------------------------------
# МЕДЛЕННО (блокирующий I/O):
(http-get url)

# БЫСТРО (coroutine):
(co 'fetch
   (http-get url)
   (yield) )

# ⚡ Не блокирует выполнение!

Совет 2: Используйте базу данных
--------------------------------
# МЕДЛЕННО (файлы):
(file-read "data.txt")

# БЫСТРО (встроенная БД):
(db:get 'key)

# ⚡ Встроенная БД быстрее!

Совет 3: Избегайте рекурсии в hot path
--------------------------------------
# МЕДЛЕННО (рекурсия):
(de sum (N)
   (if (<= N 0)
      0
      (+ N (sum (- N 1))) ) )

# БЫСТРО (итерация):
(de sum (N)
   (let Acc 0
      (for (I 1 N)
         (inc 'Acc I) )
      Acc ) )

# ⚡ Итерация быстрее рекурсии!

Совет 4: Компилируйте в bytecode
--------------------------------
# МЕДЛЕННО (интерпретация):
pil script.l

# БЫСТРО (bytecode):
pil -c script.l
pil script.lc

# ⚡ Ускорение: 2-5x!

Совет 5: Используйте встроенные функции
---------------------------------------
# МЕДЛЕННО (ручная реализация):
(de my-length (L)
   (if (null L)
      0
      (inc (my-length (cdr L))) ) )

# БЫСТРО (встроенная):
(length L)

# ⚡ Встроенные функции оптимизированы!
