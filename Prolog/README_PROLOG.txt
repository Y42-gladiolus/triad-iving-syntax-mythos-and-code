3. БЫСТРЫЙ СТАРТ
================================================================================

# Запуск интерпретатора
prolog
# или
swipl

# В интерпретаторе:
?- write('Hello, World!'), nl.
Hello, World!
true.

# Загрузка файла
?- [myfile].
% или
?- consult('myfile.pl').

# Выход
?- halt.

4. СТРУКТУРА ПРОГРАММЫ
================================================================================

% Файл: family.pl

% Факты
parent(john, mary).
parent(john, bob).
parent(mary, ann).

% Правила
grandparent(X, Z) :- parent(X, Y), parent(Y, Z).
sibling(X, Y) :- parent(P, X), parent(P, Y), X \= Y.

% Запросы (в интерпретаторе):
% ?- parent(john, X).
% X = mary ;
% X = bob.

5. ПОДВОДНЫЕ КАМНИ
================================================================================

⚠️ Рекурсия без базового случая → бесконечный цикл
⚠️ Отсечение (!) меняет логику программы
⚠️ Порядок правил важен для производительности
⚠️ Нет типизации → ошибки только при выполнении
⚠️ Backtracking может быть медленным
