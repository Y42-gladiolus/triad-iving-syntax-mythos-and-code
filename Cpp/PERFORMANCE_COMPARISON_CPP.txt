3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ C++
================================================================================

Совет 1: Используйте move semantics
-----------------------------------
// МЕДЛЕННО (copy):
std::string s1 = "hello";
std::string s2 = s1;  // Copy

// БЫСТРО (move):
std::string s1 = "hello";
std::string s2 = std::move(s1);  // Move

// ⚡ Move быстрее copy!

Совет 2: Используйте reserve для vectors
----------------------------------------
// МЕДЛЕННО (reallocation):
std::vector<int> vec;
for (int i = 0; i < 1000000; i++) {
    vec.push_back(i);
}

// БЫСТРО (reserve):
std::vector<int> vec;
vec.reserve(1000000);
for (int i = 0; i < 1000000; i++) {
    vec.push_back(i);
}

// ⚡ Reserve предотвращает reallocation!

Совет 3: Используйте const reference
------------------------------------
// МЕДЛЕННО (copy):
void print(std::string str) {
    std::cout << str << std::endl;
}

// БЫСТРО (const reference):
void print(const std::string& str) {
    std::cout << str << std::endl;
}

// ⚡ Const reference избегает copy!

Совет 4: Используйте inline и LTO
---------------------------------
// МЕДЛЕННО (без inline):
int square(int x) {
    return x * x;
}

// БЫСТРО (inline):
inline int square(int x) {
    return x * x;
}

// Compile with LTO:
g++ -O3 -flto program.cpp

// ⚡ LTO оптимизирует через translation units!

Совет 5: Используйте smart pointers
-----------------------------------
// МЕДЛЕННО (raw pointers):
int* ptr = new int[100];
// ...
delete[] ptr;  // Может забыть!

// БЫСТРО (smart pointers):
auto ptr = std::make_unique<int[]>(100);
// Автоматическое освобождение!

// ⚡ Smart pointers безопаснее!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.cpp:
-----------------------
#include <iostream>
#include <chrono>

int main() {
    auto start = std::chrono::high_resolution_clock::now();
    
    long long sum = 0;
    for (int i = 0; i < 1000000000; i++) {
        sum += i;
    }
    
    auto end = std::chrono::high_resolution_clock::now();
    auto duration = std::chrono::duration_cast<std::chrono::milliseconds>(end - start);
    
    std::cout << "Sum: " << sum << std::endl;
    std::cout << "Time: " << duration.count() << " ms" << std::endl;
    
    return 0;
}

Компиляция и запуск:
--------------------
g++ -O3 -std=c++17 benchmark.cpp -o benchmark
time ./benchmark

Сравнение с Python:
-------------------
# C++ будет в 50-100 раз быстрее!
