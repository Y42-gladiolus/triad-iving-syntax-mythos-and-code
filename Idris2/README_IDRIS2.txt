1. УСТАНОВКА IDRIS2
================================================================================

┌──────────────────────────────────────────────────────────────────────────────┐
│                    ВАЖНО: IDRIS2 ТРЕБУЕТ GHC (Haskell)                       │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  Idris2 написан на Idris1 который компилируется через Haskell               │
│  Поэтому нужен GHC (Glasgow Haskell Compiler)                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

Linux (Ubuntu/Debian):
----------------------
# Шаг 1: Установить GHC и зависимости
sudo apt update
sudo apt install ghc cabal-install git make gcc libffi-dev libgmp-dev

# Шаг 2: Обновить cabal
cabal update

# Шаг 3: Установить Idris2 через cabal
cabal install idris2

# Шаг 4: Добавить в PATH
export PATH="$HOME/.cabal/bin:$PATH"

# Проверка
idris2 --version

# Альтернатива: через пакетный менеджер (может быть старая версия)
sudo apt install idris  # Это Idris1, не рекомендуется!

macOS (Homebrew):
-----------------
brew install idris2

# Проверка
idris2 --version

Windows:
--------
# Вариант 1: WSL (рекомендуется)
# Установить WSL2 + Ubuntu, затем как Linux выше

# Вариант 2: Native (сложно)
# Следовать инструкции с https://github.com/idris-lang/Idris2

Docker (универсально):
----------------------
docker run -it --rm zohl/idris2 idris2 --version

Main> :printType "hello"

Main> 2 + 2
4 : Integer

# Выход:
Main> :quit
или :q

# Запуск скрипта:
idris2 script.idr

# Компиляция:
idris2 --build test.ipkg
idris2 --compile-to-cheap test.idr

# Создание проекта:
mkdir my_project && cd my_project
idris2 --create test.ipkg

4. УПРАВЛЕНИЕ ПАКЕТАМИ
================================================================================

# Idris2 использует .ipkg файлы для проектов

# Структура проекта:
my_project/
├── test.ipkg          # Пакет манифест
├── src/               # Исходный код
│   └── Main.idr
└── build/             # Скомпилированные файлы

# test.ipkg содержимое:
package test

modules = Main

depends = base

# Сборка проекта
idris2 --build test.ipkg

# Запуск
idris2 --run Main

# Установка пакетов из репозитория
# (пока нет центрального репозитория как opam/cargo)
# Клонировать вручную с GitHub

# Пример зависимости в .ipkg:
depends = base, contrib

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: GHC не установлен
   cabal: command not found
   
   Решение: 
   sudo apt install ghc cabal-install

⚠️ Проблема 2: Долгая первая компиляция
   Первый build может занять 30-60 минут
   
   Решение: Это нормально - компиляция стандартной библиотеки
   Используйте --compile-to-cheap для быстрой компиляции

⚠️ Проблема 3: PATH не настроен
   idris2: command not found
   
   Решение: Добавить в ~/.bashrc
   export PATH="$HOME/.cabal/bin:$PATH"

⚠️ Проблема 4: Версия Idris1 vs Idris2
   Idris1 устарел! Нужен Idris2
   
   Решение: Убедиться что установлен idris2 не idris

⚠️ Проблема 5: Зависимости не находятся
   Cannot find module
   
   Решение: Проверить depends в .ipkg файле
   depends = base, contrib

⚠️ Проблема 6: Emacs/Vim интеграция не работает
   Команды Idris не отвечают
   
   Решение: 
   - Убедиться что idris2 в PATH
   - Перезапустить редактор
   - Проверить версию idris2-mode

# Тест 1: Версия
idris2 --version

# Тест 2: Простое вычисление в REPL
echo ":printType 5" | idris2

# Тест 3: Компиляция простого файла
echo 'module Main
main : IO ()
main = putStrLn "Hello"' > test.idr
idris2 --compile-to-cheap test.idr
./test
rm test.idr test

# Если все тесты прошли → установка успешна!

cd Idris2-contrib
idris2 --install contrib.ipkg
