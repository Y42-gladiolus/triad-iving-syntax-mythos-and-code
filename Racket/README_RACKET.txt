1. УСТАНОВКА RACKET
================================================================================

Linux (Ubuntu/Debian):
----------------------
# Официальный способ (рекомендуется)
wget https://download.racket-lang.org/installers/8.12/racket-8.12-x86_64-linux-cs.sh
chmod +x racket-8.12-x86_64-linux-cs.sh
sudo ./racket-8.12-x86_64-linux-cs.sh --unix-style --dest /usr/local/racket

# Добавить в PATH
export PATH="/usr/local/racket/bin:$PATH"

# Проверка
racket --version
raco --version

# Альтернатива: через пакетный менеджер (может быть старая версия)
sudo apt install racket

macOS (Homebrew):
-----------------
brew install racket

# Проверка
racket --version

Windows:
--------
1. Скачать с https://racket-lang.org/download/
2. Запустить установщик .exe
3. Добавить в PATH при установке
4. Проверка в PowerShell: racket --version

Docker (универсально):
----------------------
docker run -it --rm racket/racket racket --version

4. УПРАВЛЕНИЕ ПАКЕТАМИ (RACO)
================================================================================

# Установка пакета
raco pkg install package-name

# Установка из Git
raco pkg install git://github.com/user/repo.git

# Просмотр установленных пакетов
raco pkg show

# Обновление пакетов
raco pkg update --all

# Удаление пакета
raco pkg remove package-name

# Создание нового пакета
raco pkg new my-package

# Тестирование пакета
raco test my-package/

# Документация
raco docs package-name

# Сборка исполняемого файла
raco exe script.rkt

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: PATH не настроен
   Команды racket/raco не найдены
   
   Решение: Добавить в ~/.bashrc
   export PATH="/usr/local/racket/bin:$PATH"

⚠️ Проблема 2: Старая версия в репозиториях
   Ubuntu может иметь версию 6.x
   
   Решение: Использовать официальный установщик

⚠️ Проблема 3: DrRacket не запускается (Linux)
   Могут отсутствовать GUI библиотеки
   
   Решение: 
   sudo apt install libgtk-3-dev

⚠️ Проблема 4: Кодировка (кириллица)
   Решение: Сохранять файлы в UTF-8

⚠️ Проблема 5: Конфликты версий пакетов
   Решение: Использовать коллекции (collections)
   raco collect --link my-collection

echo "alias rkd='raco docs'" >> ~/.zshrc

echo "alias rkt='raco test'" >> ~/.bashrc
echo "alias rkt='raco test'" >> ~/.zshrc

# Применить изменения
source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true

7. ПРОВЕРКА РАБОТОСПОСОБНОСТИ
================================================================================

# Тест 1: Версия
racket --version

# Тест 2: Простое вычисление
racket -e '(displayln (+ 2 3))'

# Тест 3: Загрузка библиотеки
racket -e '(require racket/list)'

# Тест 4: Установка и загрузка пакета
raco pkg install --batch memoize
racket -e '(require memoize)'

# Если все тесты прошли → установка успешна!
