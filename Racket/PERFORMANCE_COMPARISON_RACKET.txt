3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ RACKET
================================================================================

Совет 1: Используйте Typed Racket
---------------------------------
; МЕДЛЕННО (dynamic):
#lang racket
(define (sum lst)
  (if (null? lst) 0 (+ (car lst) (sum (cdr lst)))))

; БЫСТРО (typed):
#lang typed/racket
(: sum (-> (Listof Integer) Integer))
(define (sum lst)
  (if (null? lst) 0 (+ (car lst) (sum (cdr lst)))))

; ⚡ Ускорение: 2-10x!

Совет 2: Tail-Call Optimization
-------------------------------
; МЕДЛЕННО (не tail-recursive):
(define (sum lst)
  (if (null? lst)
      0
      (+ (car lst) (sum (cdr lst)))))  ; Stack растёт!

; БЫСТРО (tail-recursive):
(define (sum lst [acc 0])
  (if (null? lst)
      acc
      (sum (cdr lst) (+ acc (car lst)))))  ; Stack не растёт!

; ⚡ Ускорение: бесконечная рекурсия без переполнения!

Совет 3: Векторы вместо списков для доступа по индексу
------------------------------------------------------
; МЕДЛЕННО (list - O(n) доступ):
(define lst (range 10000))
(list-ref lst 5000)  ; O(n)

; БЫСТРО (vector - O(1) доступ):
(define vec (list->vector lst))
(vector-ref vec 5000)  ; O(1)

; ⚡ Ускорение: 10-100x для случайного доступа!

Совет 4: Избегайте контрактов в горячем коде
--------------------------------------------
; МЕДЛЕННО (контракты):
(define/contract (add x y)
  (-> integer? integer? integer?)
  (+ x y))

; БЫСТРО (без контрактов в Typed Racket):
#lang typed/racket
(: add (-> Integer Integer Integer))
(define (add x y)
  (+ x y))

; ⚡ Ускорение: 2-5x!

Совет 5: Используйте inexact для численных вычислений
-----------------------------------------------------
; МЕДЛЕННО (точные числа):
(define (sum n)
  (apply + (map (λ (x) (/ 1 x)) (range 1 n))))

; БЫСТРО (неточные числа):
(define (sum n)
  (apply + (map (λ (x) (/ 1.0 x)) (range 1 n))))

; ⚡ Ускорение: 5-10x для float вычислений!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.rkt:
-----------------------
#lang racket
(require racket/fixnum)

; Тест 1: Сумма
(define (sum n)
  (for/sum ([i n]) i))

(time (sum 1000000))

; Тест 2: Рекурсия (Fibonacci)
(define (fib n)
  (if (< n 2)
      n
      (+ (fib (- n 1)) (fib (- n 2)))))

(time (fib 30))

; Тест 3: Typed Racket (сравнение)
#lang typed/racket
(: sum-typed (-> Fixnum Fixnum))
(define (sum-typed n)
  (for/sum: : Fixnum ([i (in-range n)]) i))

(time (sum-typed 1000000))

Запуск:
-------
racket benchmark.rkt
