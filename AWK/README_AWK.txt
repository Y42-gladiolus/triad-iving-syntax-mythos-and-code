3. БЫСТРЫЙ СТАРТ
================================================================================

# AWK не имеет REPL (интерактивный через CLI)
# Запуск из командной строки

# Простой one-liner
echo "hello world" | awk '{print $1}'

# Из файла
awk '{print $1}' file.txt

# AWK скрипт (файл .awk)
#!/usr/bin/awk -f
{print $1}

# Запуск скрипта
awk -f script.awk file.txt

# Или через shebang
chmod +x script.awk
./script.awk file.txt

# Создание проекта
mkdir my_project && cd my_project

# Структура проекта:
my_project/
├── process.awk      # AWK скрипт
├── input.txt        # Входные данные
├── output.txt       # Выходные данные
└── run.sh           # Скрипт запуска

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Разные реализации AWK
   gawk vs mawk vs nawk могут отличаться
   
   Решение:
   Использовать gawk для совместимости
   Проверять версию: awk --version

⚠️ Проблема 2: Нет REPL
   Нельзя тестировать интерактивно
   
   Решение:
   Использовать echo для тестов
   Писать one-liners для проверки

⚠️ Проблема 3: Поля разделены пробелами
   По умолчанию FS (field separator) = пробел
   
   Решение:
   Указать -F для других разделителей
   awk -F',' для CSV

⚠️ Проблема 4: 1-indexed поля
   $1 = первое поле (не $0!)
   
   Решение:
   Запомнить: $0 = вся строка, $1 = первое поле

⚠️ Проблема 5: gawk расширения
   Не все функции работают в mawk/nawk
   
   Решение:
   Использовать POSIX AWK для совместимости
   Проверять документацию

⚠️ Проблема 6: Большие файлы
   AWK читает в память (построчно)
   
   Решение:
   AWK эффективен для больших файлов
   Использовать потоковую обработку

6. АЛИАСЫ ДЛЯ ТЕРМИНАЛА
================================================================================

# Добавить в ~/.bashrc, ~/.zshrc, ~/.mkshrc

echo "alias awk='gawk'" >> ~/.bashrc
echo "alias awk='gawk'" >> ~/.zshrc

echo "alias awkf='awk -f'" >> ~/.bashrc
echo "alias awkf='awk -f'" >> ~/.zshrc

echo "alias awkF='awk -F'" >> ~/.bashrc
echo "alias awkF='awk -F'" >> ~/.zshrc

# Применить изменения
source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true

# Тест 1: Версия AWK

# Тест 3: Обработка файла
echo -e "1\n2\n3" | awk '{sum+=$1} END {print sum}'

# Тест 4: CSV обработка
echo -e "a,b,c\n1,2,3" | awk -F',' '{print $1}'
