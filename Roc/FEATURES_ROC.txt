1. АРХИТЕКТУРА И ДИЗАЙН
================================================================================

┌──────────────────────────────────────────────────────────────────────────────┐
│                    ROC ARCHITECTURE                                          │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  IMPLEMENTATION:                                                             │
│  • Написан на Zig (быстрый, безопасный)                                     │
│  • Компилируется в LLVM IR                                                  │
│  • Оптимизации уровня production                                            │
│  • Параллельная компиляция                                                  │
│                                                                              │

│  • Fast compilation (быстрее Haskell/OCaml)                                 │
│  • Friendly error messages (лучший UX)                                      │
│  • No garbage collector (reference counting)                                │
│  • Functional programming (без type classes)                                │
│  • Web-first (WebAssembly + native)                                         │
│                                                                              │
│  KEY FEATURES:                                                               │
│  • Built-in formatting (roc format)                                         │
│  • Type inference (вывод типов)                                             │
│  • No null (как Elm)                                                        │
│  • Result types для ошибок                                                  │
│  • Abilities (новая концепция вместо type classes)                          │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

│  • Fast inference                                                           │

5. ERROR HANDLING
================================================================================

┌──────────────────────────────────────────────────────────────────────────────┐
│                    ROC ERROR HANDLING                                        │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  RESULT TYPES:                                                               │
│  • Result ok err - для ошибок                                               │
│  • Нет exceptions                                                           │
│  • Все ошибки явные в типах                                                 │
│  • Как Rust, Elm                                                            │
│                                                                              │
│  EXAMPLES:                                                                   │
│  • Result Str [NotFound, PermissionDenied]                                  │
│  • Ok "success"                                                             │
│  • Err NotFound                                                             │
│                                                                              │
│  PATTERN MATCHING:                                                           │
│  • Когда for handling results                                               │
│  • Exhaustive checking                                                      │
│  • Компилятор проверяет все cases                                           │
│                                                                              │
│  EXAMPLE:                                                                    │
│  when readFile "file.txt" is                                                │
│      Ok content -> Stdout.line content                                      │
│      Err NotFound -> Stdout.line "File not found"                           │
│      Err PermissionDenied -> Stdout.line "Permission denied"                │
│                                                                              │
│  NO EXCEPTIONS:                                                              │
│  • Нет try/catch                                                            │
│  • Все ошибки через Result                                                  │
│  • Нет runtime exceptions                                                   │
│  • Все ошибки compile-time checked                                          │
│                                                                              │
│  COMPILER ERRORS:                                                            │
│  • Отличные сообщения об ошибках                                            │
│  • Цветной вывод                                                            │
│  • Предложения по исправлению                                               │
│  • Лучший compiler UX в индустрии                                           │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘
