Linux (Ubuntu/Debian):
----------------------
# Вариант 1: Через официальный скрипт
curl -fL https://github.com/roc-lang/roc/releases/latest/download/roc_nightly-linux_x86_64.tar.gz | tar -xz
sudo mv roc /usr/local/bin/

# Вариант 2: Через cargo (если есть Rust)
cargo install roc

# Проверка
roc version

macOS (Homebrew):
-----------------
brew install roc

# Проверка
roc version

Windows:
--------
# Скачать с https://github.com/roc-lang/roc/releases
# Распаковать архив
# Добавить в PATH

# Проверка в PowerShell:
roc version

Docker (универсально):
----------------------
docker run -it --rm ghcr.io/roc-lang/roc:latest roc version

================================================================================
3. БЫСТРЫЙ СТАРТ
================================================================================

# Roc REPL
roc repl

# В REPL:
» 2 + 2
4 : Num *

# Создание проекта
roc init my_project
cd my_project

# Структура проекта:
my_project/
├── main.roc         # Главный файл
├── roc.yaml         # Конфигурация проекта
└── packages/        # Зависимости

# Простая программа (main.roc)
app "hello"
    packages { pf: "https://github.com/roc-lang/basic-cli/releases/latest/download/basic-cli.tar.br" }
    imports [pf.Stdout]
    provides [main] to pf

main =
    Stdout.line "Hello, Roc!"

# Запуск
roc run main.roc

# Компиляция в executable
roc build main.roc

# Компиляция в WebAssembly
roc build --target=wasm32 main.roc

# Форматирование
roc format main.roc

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Очень новый язык
   Roc в активной разработке (alpha/beta)
   
   Решение:
   Следить за обновлениями
   Использовать nightly builds

⚠️ Проблема 2: Меньше библиотек
   Экосистема только формируется
   
   Решение:
   Использовать FFI для C/JS библиотек
   Писать свои библиотеки

⚠️ Проблема 3: Документация
   Документация ещё развивается
   
   Решение:
   Читать roc-lang.org
   Присоединиться к Discord

⚠️ Проблема 4: IDE поддержка
   Меньше чем mainstream языки
   
   Решение:
   Использовать VS Code + Roc extension
   Или Zed editor

⚠️ Проблема 5: Breaking changes
   Язык быстро развивается
   
   Решение:
   Использовать specific versions
   Следить за changelog

⚠️ Проблема 6: Сообщество
   Маленькое но активное
   
   Решение:
   Присоединиться к Roc Discord
   Читать GitHub discussions
