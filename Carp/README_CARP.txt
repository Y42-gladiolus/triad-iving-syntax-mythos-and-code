┌──────────────────────────────────────────────────────────────────────────────┐
│                    CARP = LISP + C + НЕТ GARBAGE COLLECTOR                   │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  Carp используется:                                                          │
│  • Real-time системы                                                        │
│  • Игры (game development)                                                  │
│  • Embedded системы                                                         │
│  • Audio/Video обработка                                                    │
│  • Когда нужен Lisp но без GC                                               │
│                                                                              │
│  • Компилируется в C                                                        │
│  • Статическая типизация                                                    │
│  • Нет garbage collector                                                    │
│  • Lisp синтаксис (S-выражения)                                             │
│  • 2016+ дизайн (современный)                                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

Linux (Ubuntu/Debian):
----------------------
# Вариант 1: Через stack (Haskell)
sudo apt install stack
stack install carp

# Вариант 2: Из исходников (рекомендуется!)
git clone https://github.com/carp-lang/Carp.git
cd Carp
stack install
# или
cabal install

# Вариант 3: Бинарный релиз
wget https://github.com/carp-lang/Carp/releases/download/v0.5.2/carp-linux-v0.5.2.zip
unzip carp-linux-v0.5.2.zip
sudo cp carp /usr/local/bin/

# Проверка
carp --version

macOS (Homebrew):
-----------------
brew install carp-lang/carp/carp

# Проверка
carp --version

Windows:
--------
# Требуется WSL или Stack
stack install carp

# Проверка в PowerShell:
carp --version

Docker (универсально):
----------------------
docker run -it --rm carp-lang/carp carp --version

3. БЫСТРЫЙ СТАРТ
================================================================================

# Carp REPL
carp

# В REPL:
Carp> (+ 2 2)
4 : Int
Carp> (quit)  # Выход

# Запуск скрипта
carp script.carp

# Запуск с кодом из командной строки
carp -e "(+ 2 2)"

# Компиляция в C
carp --compile script.carp
gcc -o script script.c

# Создание проекта
mkdir my_project && cd my_project

# Структура проекта:
my_project/
├── Main.carp      # Главный файл
├── Lib/           # Библиотеки
│   └── Utils.carp
├── test/          # Тесты
└── README.md      # Документация

4. УПРАВЛЕНИЕ ПАКЕТАМИ
================================================================================

# Carp имеет пакетный менеджер (Carp Packages)

# Поиск пакетов
carp pkg search package_name

# Установка пакета
carp pkg install package_name

# Обновление пакетов
carp pkg update

# Carp Package Repository:
# https://github.com/carp-lang/packages

# Структура .carp файла:
(module Main)

(defn add [a b]
  (+ a b))

# Использование
(load "Lib/Utils.carp")
(add 5 3)

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Требует Haskell (stack/cabal)
   Carp написан на Haskell
   
   Решение:
   stack setup
   # Или установить GHC

⚠️ Проблема 2: Нет GC (ручное управление памятью)
   Непривычно для Lisp-еров
   
   Решение:
   Изучить модель памяти Carp
   Читать документацию по ownership

⚠️ Проблема 3: Статическая типизация
   Для Lisp-еров это ново
   
   Решение:
   Читать документацию по типам
   Практиковаться ежедневно

⚠️ Проблема 4: Мало документации
   Молодой проект (2016+)
   
   Решение:
   Читать официальную документацию
   Изучать примеры на GitHub

⚠️ Проблема 5: Компиляция в C
   Требуется GCC/Clang
   
   Решение:
   sudo apt install build-essential

⚠️ Проблема 6: Разные версии Carp
   Быстро развивается
   
   Решение:
   Использовать стабильные релизы
   Следить за обновлениями

echo "alias cpc='carp --compile'" >> ~/.zshrc

echo "alias cpe='carp -e'" >> ~/.zshrc

# Применить изменения
source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true

7. ПРОВЕРКА РАБОТОСПОСОБНОСТИ
================================================================================

# Тест 1: Версия Carp
carp --version

# Тест 2: Простое вычисление
echo "(+ 2 2)" | carp

# Тест 3: Запуск скрипта
echo '(defn main [] (println "OK"))' > test.carp
carp test.carp
rm test.carp

# Тест 4: Компиляция в C
echo '(defn main [] (println "OK"))' > test.carp
carp --compile test.carp
./test
rm test.carp test.c test

# Если все тесты прошли → установка успешна!
