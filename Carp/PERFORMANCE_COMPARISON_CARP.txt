3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ CARP
================================================================================

Совет 1: Используйте static arrays
----------------------------------
; МЕДЛЕННО (dynamic array):
(def arr (array 1 2 3))

; БЫСТРО (static array):
(def arr (static-array Int 3))

; ⚡ Static arrays быстрее!

Совет 2: Избегайте аллокаций в циклах
-------------------------------------
; МЕДЛЕННО (аллокация в цикле):
(loop [i 0]
  (let [arr (array i)]
    (use arr)
    (array-free arr)))

; БЫСТРО (переиспользование):
(let [arr (array 0)]
  (loop [i 0]
    (array-set arr 0 i)
    (use arr)))

; ⚡ Меньше аллокаций = быстрее!

Совет 3: Используйте tail-рекурсию
----------------------------------
; МЕДЛЕННО (обычная рекурсия):
(defn sum [n]
  (if (<= n 0)
    0
    (+ n (sum (- n 1)))))

; БЫСТРО (tail-рекурсия):
(defn sum-tail [n acc]
  (if (<= n 0)
    acc
    (recur (- n 1) (+ acc n))))

; ⚡ Carp оптимизирует tail-рекурсию!

Совет 4: Компилируйте с оптимизациями
-------------------------------------
; МЕДЛЕННО (debug):
carp --compile script.carp

; БЫСТРО (release):
carp --compile -O3 script.carp

; ⚡ Ускорение: 5-10x для runtime!

Совет 5: Используйте C функции для hot path
-------------------------------------------
; МЕДЛЕННО (Carp):
(defn sum [arr]
  (array-reduce + 0 arr))

; БЫСТРО (C):
(extern "stdlib.h"
  (defn c-sum [arr len] : Int))

; ⚡ C функции быстрее!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.carp:
------------------------
(defn sum-enum [n]
  (loop [i 0 acc 0]
    (if (>= i n)
      acc
      (recur (+ i 1) (+ acc i)))))

(def start (time-now))
(sum-enum 100000000)
(def end (time-now))

(println "Time: " (- end start) " ms")

Запуск:
-------
carp --compile -O3 benchmark.carp
./benchmark

Сравнение с Racket:
-------------------
# Carp будет в 10-20 раз быстрее!
# Нет GC оверхеда!
