5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Разные shell реализации
   mksh vs bash vs dash vs sh могут отличаться
   
   Решение:
   Использовать #!/bin/mksh для mksh
   Использовать #!/bin/sh для POSIX

⚠️ Проблема 2: bash-измы в mksh
   Некоторые bash функции не работают в mksh
   
   Решение:
   Проверять POSIX совместимость
   Использовать shellcheck для проверки

⚠️ Проблема 3: Переменные окружения
   Экспорт переменных может не работать
   
   Решение:
   export VAR=value  # Явный export

⚠️ Проблема 4: Арифметика
   mksh арифметика отличается от bash
   
   Решение:
   Использовать $(( )) для арифметики
   Избегать bash специфичных конструкций

⚠️ Проблема 5: Arrays
   mksh arrays отличаются от bash
   
   Решение:
   Использовать POSIX совместимые arrays
   Или избегать arrays если возможно

⚠️ Проблема 6: PATH не настроен
   mksh: command not found
   
   Решение:
   export PATH=/usr/local/bin:$PATH
   Добавить в ~/.mkshrc

# Добавить в ~/.bashrc, ~/.zshrc, ~/.mkshrc

echo "alias mk='mksh'" >> ~/.bashrc
echo "alias mk='mksh'" >> ~/.zshrc

echo "alias mkshrc='vi ~/.mkshrc'" >> ~/.bashrc
echo "alias mkshrc='vi ~/.mkshrc'" >> ~/.zshrc

# Применить изменения
source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true
