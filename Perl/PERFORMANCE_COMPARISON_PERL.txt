3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ PERL
================================================================================

Совет 1: Используйте -n и -p флаги
----------------------------------
# МЕДЛЕННО (явный цикл):
while (<>) {
    print if /pattern/;
}

# БЫСТРО (флаг -n):
perl -ne 'print if /pattern/'

# ⚡ -n и -p экономят код!

Совет 2: Компилируйте regex
---------------------------
# МЕДЛЕННО (компиляция каждый раз):
while (<>) {
    if (/pattern/) {
        print;
    }
}

# БЫСТРО (компиляция один раз):
my $regex = qr/pattern/;
while (<>) {
    if (/$regex/) {
        print;
    }
}

# ⚡ qr// компилирует regex один раз!

Совет 3: Используйте map/grep
-----------------------------
# МЕДЛЕННО (цикл):
my @result;
foreach my $item (@array) {
    push @result, $item * 2 if $item > 0;
}

# БЫСТРО (map/grep):
my @result = map { $_ * 2 } grep { $_ > 0 } @array;

# ⚡ map/grep быстрее и выразительнее!

Совет 4: Избегайте лишних копий
-------------------------------
# МЕДЛЕННО (копирование):
my @copy = @array;
foreach my $item (@copy) {
    $item++;
}

# БЫСТРО (ссылка):
foreach my $item (@array) {
    $item++;
}

# ⚡ Работайте с оригиналом когда возможно!

Совет 5: Используйте slurp mode для маленьких файлов
---------------------------------------------------
# МЕДЛЕННО (построчно):
my @lines;
while (<>) {
    push @lines, $_;
}

# БЫСТРО (slurp):
my $content = do { local $/; <> };

# ⚡ Slurp быстрее для маленьких файлов!

8. ИНТЕГРАЦИЯ PERL С ДРУГИМИ ЯЗЫКАМИ
================================================================================

Perl + bash (pipeline):
-----------------------
cat file.txt | perl -lane 'print $F[0]' | sort | uniq

Perl + Python (через stdin/stdout):
-----------------------------------
perl -lane 'print $F[0]' file.txt | python3 process.py

Perl + AWK (комбинация):
------------------------
awk '{print $1}' file.txt | perl -lane 'print uc $F[0]'

Perl + MiniZinc (через CLI):
----------------------------
perl -lane 'print $F[0]' data.txt | minizinc model.mzn
