3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ LFE
================================================================================

Совет 1: Используйте tail recursion
-----------------------------------
; МЕДЛЕННО (обычная рекурсия):
(defun sum (list)
  (if (null list)
    0
    (+ (car list) (sum (cdr list)))))

; БЫСТРО (tail recursion):
(defun sum (list)
  (sum-helper list 0))

(defun sum-helper (list acc)
  (if (null list)
    acc
    (sum-helper (cdr list) (+ acc (car list)))))

; ⚡ Tail recursion оптимизирована!

Совет 2: Используйте binary для данных
--------------------------------------
; МЕДЛЕННО (list):
(data := '(1 2 3 4 5))

; БЫСТРО (binary):
(data := <<1,2,3,4,5>>)

; ⚡ Binary эффективнее для данных!

Совет 3: Используйте process dictionary осторожно
-------------------------------------------------
; МЕДЛЕННО (get/put):
(put 'key value)
(get 'key)

; БЫСТРО (state passing):
(defun process (state data)
  (let ((new-state (update state data)))
    {ok, new-state}))

; ⚡ State passing лучше для OTP!

Совет 4: Используйте supervisor trees
-------------------------------------
; МЕДЛЕННО (manual supervision):
(spawn (lambda ()
  (try
    (work)
  (catch
    (_ _)
      (restart)))))

; БЫСТРО (supervisor):
(supervisor:start_link 'my_supervisor [])

; ⚡ Supervisor trees надёжнее!

Совет 5: Используйте hot code reloading
---------------------------------------
; МЕДЛЕННО (restart):
; Остановить сервер
; Запустить новую версию

; БЫСТРО (hot reload):
c(my_module).  ; В REPL

; ⚡ Hot code reloading без downtime!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.lfe:
-----------------------
(defmodule benchmark
  (export all))

(defun sum (n)
  (sum-helper n 0))

(defun sum-helper (0 acc)
  acc)
(defun sum-helper (n acc)
  (sum-helper (- n 1) (+ acc n)))

(defun main (args)
  (let ((start (erlang:system_info 'wall_clock)))
    (sum 1000000000)
    (let ((end (erlang:system_info 'wall_clock)))
      (io:format "Time: ~p ms~n" (list (- end start))))))

Запуск:
-------
rebar3 compile
rebar3 run
