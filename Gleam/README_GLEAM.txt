1. УСТАНОВКА GLEAM
================================================================================

┌──────────────────────────────────────────────────────────────────────────────┐
│                    GLEAM = СТАТИЧЕСКАЯ ТИПИЗАЦИЯ ДЛЯ BEAM                    │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  Gleam компилируется в:                                                      │
│  • Erlang/Elixir (BEAM VM) — для backend                                    │
│  • JavaScript/TypeScript — для frontend                                     │
│                                                                              │
│  Это как "типизированный Elixir" с лучшим compiler!                         │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

Linux (Ubuntu/Debian):
----------------------
# Вариант 1: Через официальный установщик
curl -fsSL https://gleam.run/install.sh | sh

# Вариант 2: Через asdf (если уже установлен)
asdf plugin-add gleam
asdf install gleam latest
asdf global gleam latest

# Вариант 3: Через Cargo (Rust)
cargo install gleam

# Проверка
gleam --version

macOS (Homebrew):
-----------------
brew install gleam

# Проверка
gleam --version

Windows:
--------
# Вариант 1: Через winget
winget install gleam

# Вариант 2: Через Chocolatey
choco install gleam

# Вариант 3: WSL (рекомендуется)
# Установить WSL2 + Ubuntu, затем как Linux выше

Docker (универсально):
----------------------
docker run -it --rm ghcr.io/gleam-lang/gleam gleam --version

================================================================================
3. БЫСТРЫЙ СТАРТ
================================================================================

# Создание нового проекта
gleam new my_project
cd my_project

# Структура проекта:
my_project/
├── gleam.toml         # Конфигурация проекта
├── manifest.toml      # Заблокированные версии
├── src/               # Исходный код
│   └── my_project.gleam
├── test/              # Тесты
│   └── my_project_test.gleam
└── build/             # Скомпилированные файлы

# Запуск проекта
gleam run

# Запуск тестов
gleam test

# Форматирование
gleam format

# Документация
gleam docs build

# Компиляция в Erlang
gleam build --target erlang

# Компиляция в JavaScript
gleam build --target javascript

# REPL (Shell)
gleam shell

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Erlang не установлен
   Gleam требует Erlang/OTP для BEAM target
   
   Решение: 
   asdf plugin-add erlang
   asdf install erlang 26.2.0

⚠️ Проблема 2: Node.js не установлен (для JS target)
   Требуется для JavaScript компиляции
   
   Решение:
   asdf plugin-add nodejs
   asdf install nodejs latest

⚠️ Проблема 3: Конфликт версий
   Разные проекты требуют разные версии
   
   Решение: asdf manage версии
   asdf local gleam 1.0.0  # Для конкретного проекта

⚠️ Проблема 4: LSP не работает в VS Code
   Автодополнение не работает
   
   Решение:
   gleam update
   Перезапустить VS Code

⚠️ Проблема 5: Кодировка (Windows)
   Кириллица может не отображаться
   
   Решение:
   Использовать WSL или настроить UTF-8
