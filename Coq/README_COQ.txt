Linux (Ubuntu/Debian):
----------------------
# Шаг 1: Установить opam
sudo apt update
sudo apt install opam

# Шаг 2: Инициализировать opam
opam init --disable-sandboxing
eval $(opam env)

# Шаг 3: Установить Coq
opam install coq

# Шаг 4: Установить Coq Platform (рекомендуется!)
opam install coq-platform

# Проверка
coqc --version
coqtop --version

macOS (Homebrew):
-----------------
brew install coq
brew install coq-platform

# Проверка
coqc --version

Windows:
--------
# Вариант 1: WSL (рекомендуется)
# Установить WSL2 + Ubuntu, затем как Linux выше

# Вариант 2: Native
# Скачать с https://coq.inria.fr/download

Docker (универсально):
----------------------
docker run -it --rm coqorg/coq coqc --version

================================================================================
3. БЫСТРЫЙ СТАРТ
================================================================================

# Запуск интерпретатора (REPL)
coqtop

# Вы увидите:
# Welcome to Coq 8.18.0
# 
# Coq <

# Первый код:
Coq < Check 2 + 2.
     = 4
     : nat

Coq < Theorem add_comm: forall n m : nat, n + m = m + n.
Coq < Proof.
Coq < intros.
Coq < rewrite Nat.add_comm.
Coq < reflexivity.
Coq < Qed.

# Выход:
Coq < Quit.

# Компиляция файла:
coqc file.v      # Компилирует в .vo

# Запуск файла:
coqtop -l file.v

# С VS Code:
# Откройте .v файл, нажмите Alt+N для следующей команды

4. УПРАВЛЕНИЕ ПАКЕТАМИ
================================================================================

# Coq использует opam для пакетов

# Установка пакетов:
opam install coq-mathcomp        # Mathematical Components
opam install coq-stdpp           # Стандартная библиотека++
opam install coq-equations       # Уравнения
opam install coq-hierarchy-builder

# Mathlib (математическая библиотека Coq):
opam install coq-math-classes

# Просмотр установленных пакетов:
opam list | grep coq

# Обновление:
opam update
opam upgrade

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: opam env не настроен
   coqc: command not found
   
   Решение: 
   eval $(opam env)
   Добавить в ~/.bashrc

⚠️ Проблема 2: Версии не совпадают
   Error: Incompatible .vo files
   
   Решение:
   Убедиться что все пакеты одной версии
   opam install coq.8.18.0

⚠️ Проблема 3: VS Code extension не работает
   Coq сервер не запускается
   
   Решение:
   - Проверить что coqc в PATH
   - Перезапустить VS Code
   - Проверить Output → Coq Platform

⚠️ Проблема 4: Долгая компиляция
   Первая компиляция Mathlib может занять 1-2 часа
   
   Решение: Это нормально - компиляция библиотеки
   Используйте кэш: ~/.opam

⚠️ Проблема 5: Proof General не загружается
   Emacs не находит Proof General
   
   Решение:
   opam install coq
   Проверить путь в .emacs

source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true
