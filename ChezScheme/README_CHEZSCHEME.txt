Linux (Ubuntu/Debian):
----------------------
# Вариант 1: Через apt (может быть старая версия)
sudo apt update
sudo apt install chez-scheme

# Вариант 2: Из исходников (рекомендуется!)
wget https://github.com/cisco/ChezScheme/releases/download/v9.5.8/chezscheme-v9.5.8.tar.gz
tar xzf chezscheme-v9.5.8.tar.gz
cd chezscheme-v9.5.8
./configure
make
sudo make install

# Вариант 3: Через пакетный менеджер
sudo apt install petite-chezscheme

# Проверка
scheme --version
# или
petite --version

macOS (Homebrew):
-----------------
brew install chez-scheme

# Проверка
scheme --version

Windows:
--------
# Скачать с https://cisco.github.io/ChezScheme/
# Установщик .exe
# Добавить в PATH: C:\Program Files\Chez Scheme

# Проверка в PowerShell:
scheme --version

Docker (универсально):
----------------------
docker run -it --rm ghcr.io/cisco/chez-scheme scheme

3. БЫСТРЫЙ СТАРТ
================================================================================

# Chez Scheme REPL
scheme

# В REPL:
> (+ 2 2)
4
> (exit)  # Выход

# Petite Chez Scheme (меньше, быстрее startup)
petite

# Запуск скрипта
scheme --script script.scm

# Запуск с кодом из командной строки
scheme --eval '(+ 2 2)'

# Компиляция в native code
scheme --compile script.scm

# Создание проекта

4. УПРАВЛЕНИЕ ПАКЕТАМИ

# Chez Scheme не имеет центрального пакетного менеджера
# Библиотеки = .scm файлы

# Поиск библиотек:
# https://github.com/scheme-community
# https://srfi.schemers.org/

# Подключение библиотек
(load "lib/utils.scm")

# Структура библиотеки:
; lib/utils.scm
(define (add a b)
  (+ a b))

# Использование
(load "lib/utils.scm")
(add 5 3)  ; 8

# SRFI (Scheme Requests for Implementation)
; Стандартные библиотеки
(import (srfi :1 lists))
(import (srfi :13 strings))

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: Две версии (Chez vs Petite)
   scheme vs petite могут отличаться
   
   Решение:
   scheme = полная версия
   petite = меньшая, быстрее startup

⚠️ Проблема 2: Разные диалекты Scheme
   R5RS, R6RS, R7RS стандарты
   
   Решение:
   Chez поддерживает R6RS по умолчанию
   Использовать (import (rnrs)) для совместимости

⚠️ Проблема 3: Мало пакетного менеджера
   Нет как npm/pip
   
   Решение:
   Использовать SRFI библиотеки
   Загружать вручную через load

⚠️ Проблема 4: Синтаксис непривычен
   S-выражения, префиксная нотация
   
   Решение:
   Практиковаться ежедневно
   Использовать REPL

⚠️ Проблема 5: Компиляция сложна
   Требуется понимание компилятора
   
   Решение:
   Читать документацию
   Изучать примеры

⚠️ Проблема 6: Разные реализации Scheme
   Chez, Racket, Guile, etc.
   
   Решение:
   Использовать SRFI для совместимости
   Проверять совместимость

echo "alias sche='scheme --eval'" >> ~/.zshrc

source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true
