3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ CHEZ SCHEME
================================================================================

Совет 1: Используйте tail-рекурсию
----------------------------------
; МЕДЛЕННО (обычная рекурсия):
(define (sum n)
  (if (<= n 0)
      0
      (+ n (sum (- n 1)))))

; БЫСТРО (tail-рекурсия):
(define (sum-tail n acc)
  (if (<= n 0)
      acc
      (sum-tail (- n 1) (+ acc n))))

; ⚡ Chez оптимизирует tail-рекурсию!

Совет 2: Компилируйте в native code
-----------------------------------
; МЕДЛЕННО (интерпретация):
scheme --script script.scm

; БЫСТРО (компиляция):
scheme --compile script.scm

; ⚡ Ускорение: 5-10x для runtime!

Совет 3: Используйте inlining
-----------------------------
; МЕДЛЕННО (вызов функции):
(define (add a b) (+ a b))

; БЫСТРО (inline):
(define add (lambda (a b) (+ a b)))
(optimize-level 3)

; ⚡ Компилятор inline'ит автоматически!

Совет 4: Избегайте cons в циклах
--------------------------------
; МЕДЛЕННО (cons в цикле):
(define (make-list n)
  (if (= n 0)
      '()
      (cons n (make-list (- n 1)))))

; БЫСТРО (vector):
(define (make-vector n)
  (let ((v (make-vector n)))
    (let loop ((i 0))
      (when (< i n)
        (vector-set! v i i)
        (loop (+ i 1))))
    v))

; ⚡ Vectors быстрее lists!

Совет 5: Используйте SRFI библиотеки
------------------------------------
; МЕДЛЕННО (ручная реализация):
(define (my-map f lst)
  (if (null? lst)
      '()
      (cons (f (car lst)) (my-map f (cdr lst)))))

; БЫСТРО (SRFI):
(import (srfi :1 lists))
(map f lst)

; ⚡ SRFI оптимизированы!

================================================================================

Создайте benchmark.scm:
-----------------------
(define (sum-enum n)
  (let loop ((i 0) (acc 0))
    (if (> i n)
        acc
        (loop (+ i 1) (+ acc i)))))

(define start (get-time))
(sum-enum 100000000)
(define end (get-time))

(display "Time: ")
(display (- end start))
(display " ms")
(newline)

Запуск:

scheme --script benchmark.scm
