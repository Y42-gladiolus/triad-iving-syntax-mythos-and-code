## 🧭 Quick Start: The Triad Table

| Layer | File Pattern | Purpose | Style | For Whom |
|-------|-------------|---------|-------|----------|
| 🔹 Reference | `01_*`, `*-code.org` | Syntax, specs, invariants | Code-first, minimal | Engineers, auditors |
| 🔹 Fable | `*-fable.org`, `02_*` | Philosophy, context, journey | Narrative, honest | Developers seeking *why* |
| 🔹 Praxis | `03_*`, `*-praxis/` | Working code, examples, tests | Executable, verified | Implementers, testers |
| 🔹 Easter Egg | `99_*`, `*-illuminati*` | Aesthetic layer, wink to elders | Mystical, intentional flaws | ~10 Smalltalk elders 😊 |

> 📌 **Rule**: Open a concept in all three layers. Read in parallel.
