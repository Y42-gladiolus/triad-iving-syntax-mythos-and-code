3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ PYTHON
================================================================================

Совет 1: Используйте NumPy для чисел
------------------------------------
# МЕДЛЕННО (чистый Python):
result = []
for i in range(1000000):
    result.append(i * 2)

# БЫСТРО (NumPy):
import numpy as np
arr = np.arange(1000000)
result = arr * 2

# ⚡ Ускорение: 50-100x!

Совет 2: Используйте list comprehension
---------------------------------------
# МЕДЛЕННО (цикл):
result = []
for i in range(1000):
    result.append(i * 2)

# БЫСТРО (comprehension):
result = [i * 2 for i in range(1000)]

# ⚡ Ускорение: 2-3x!

Совет 3: Используйте asyncio для I/O
------------------------------------
# МЕДЛЕННО (последовательно):
for url in urls:
    response = requests.get(url)

# БЫСТРО (параллельно):
import aiohttp
async with aiohttp.ClientSession() as session:
    tasks = [fetch(session, url) for url in urls]
    results = await asyncio.gather(*tasks)

# ⚡ Ускорение: 5-10x для I/O!

Совет 4: Используйте C расширения
---------------------------------
# МЕДЛЕННО (чистый Python):
def heavy_computation():
    # ...

# БЫСТРО (C расширение):
# Используйте NumPy, Cython, или Numba
import numba
@numba.jit
def heavy_computation():
    # ...

# ⚡ Ускорение: 10-100x!

Совет 5: Используйте multiprocessing для CPU-bound
--------------------------------------------------
# МЕДЛЕННО (threading, GIL ограничивает):
from threading import Thread

# БЫСТРО (multiprocessing):
from multiprocessing import Process

# ⚡ Обходит GIL!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.py:
----------------------
import time

def sum_enum(n):
    total = 0
    for i in range(1, n + 1):
        total += i
    return total

# NumPy версия
import numpy as np

def sum_numpy(n):
    return np.sum(np.arange(1, n + 1))

start = time.time()
result = sum_enum(100_000_000)
end = time.time()
print(f"Pure Python: {end - start:.3f} seconds")

start = time.time()
result = sum_numpy(100_000_000)
end = time.time()
print(f"NumPy: {end - start:.3f} seconds")

Запуск:
-------
python benchmark.py

Сравнение с другими:
--------------------
# Pure Python будет медленнее C/Rust
# NumPy будет сопоставим с C!
