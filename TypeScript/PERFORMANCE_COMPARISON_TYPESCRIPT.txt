3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ TYPESCRIPT
================================================================================

Совет 1: Используйте strict mode
--------------------------------
// tsconfig.json
{
  "compilerOptions": {
    "strict": true,
    "noImplicitAny": true,
    "strictNullChecks": true
  }
}

// ⚡ Ловит больше ошибок на compile-time!

Совет 2: Избегайте any
----------------------
// МЕДЛЕННО (любая ошибка runtime):
function process(data: any) {
  return data.length;
}

// БЫСТРО (compile-time проверка):
function process(data: string[] | string) {
  return Array.isArray(data) ? data.length : data.length;
}

// ⚡ Меньше runtime ошибок!

Совет 3: Используйте const вместо let
-------------------------------------
// МЕДЛЕННО (может измениться):
let x = 5;

// БЫСТРО (оптимизация компилятора):
const x = 5;

// ⚡ Лучше оптимизация V8!

Совет 4: Избегайте лишних аллокаций
-----------------------------------
// МЕДЛЕННО (создаёт новый массив):
const result = arr.map(x => x).filter(x => x > 0);

// БЫСТРО (один проход):
const result = arr.reduce((acc, x) => {
  if (x > 0) acc.push(x);
  return acc;
}, []);

// ⚡ Меньше аллокаций памяти!

Совет 5: Используйте Web Workers для CPU-bound
----------------------------------------------
// МЕДЛЕННО (блокирует UI):
const result = heavyComputation();

// БЫСТРО (отдельный поток):
const worker = new Worker('worker.js');
worker.postMessage(data);
worker.onmessage = (e) => console.log(e.data);

// ⚡ UI не блокируется!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.ts:
----------------------
function sumEnum(n: number): number {
  let sum = 0;
  for (let i = 1; i <= n; i++) {
    sum += i;
  }
  return sum;
}

const start = performance.now();
const result = sumEnum(1_000_000);
const end = performance.now();

console.log(`Sum: ${result}`);
console.log(`Time: ${end - start} ms`);

Компиляция и запуск:
--------------------
npx tsc benchmark.ts
node benchmark.js

# Или напрямую:
npx ts-node benchmark.ts

Сравнение с другими:
--------------------
# TypeScript будет быстрее Python, медленнее Rust
# Но экосистема больше всех!
