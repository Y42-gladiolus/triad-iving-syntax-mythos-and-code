3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ SCALA
================================================================================

Совет 1: Используйте val вместо var
-----------------------------------
// МЕДЛЕННО (mutable):
var sum = 0
for (i <- 1 to 1000) {
  sum += i
}

// БЫСТРО (immutable):
val sum = (1 to 1000).sum

// ⚡ Immutable быстрее и безопаснее!

Совет 2: Используйте collections правильно
------------------------------------------
// МЕДЛЕННО (List для random access):
val list = List(1, 2, 3, 4, 5)
list(4)  // O(n)

// БЫСТРО (Vector для random access):
val vector = Vector(1, 2, 3, 4, 5)
vector(4)  // O(1)

// ⚡ Правильные collections важны!

Совет 3: Используйте @tailrec
-----------------------------
// МЕДЛЕННО (без tailrec):
def factorial(n: Int): Int = {
  if (n <= 1) 1 else n * factorial(n - 1)
}

// БЫСТРО (с tailrec):
@tailrec
def factorial(n: Int, acc: Int = 1): Int = {
  if (n <= 1) acc else factorial(n - 1, acc * n)
}

// ⚡ Tailrec предотвращает stack overflow!

Совет 4: Используйте view/lazy evaluation
-----------------------------------------
// МЕДЛЕННО (eager evaluation):
List(1, 2, 3, 4, 5)
  .map(_ * 2)
  .filter(_ > 5)
  .head

// БЫСТРО (lazy evaluation):
List(1, 2, 3, 4, 5).view
  .map(_ * 2)
  .filter(_ > 5)
  .head

// ⚡ View избегает промежуточных коллекций!

Совет 5: Используйте параллельные коллекции
-------------------------------------------
// МЕДЛЕННО (последовательно):
List(1, 2, 3, 4, 5).map(expensiveOperation)

// БЫСТРО (параллельно):
List(1, 2, 3, 4, 5).par.map(expensiveOperation)

// ⚡ Параллельные коллекции используют multi-core!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте Benchmark.scala:
-------------------------
object Benchmark {
  def main(args: Array[String]): Unit = {
    val start = System.currentTimeMillis()
    
    var sum = 0L
    for (i <- 1 to 1000000000L) {
      sum += i
    }
    
    val end = System.currentTimeMillis()
    println(s"Sum: $sum")
    println(s"Time: ${end - start} ms")
  }
}

Компиляция и запуск:
--------------------
scalac Benchmark.scala
scala Benchmark
