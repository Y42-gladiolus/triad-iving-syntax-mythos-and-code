3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ C
================================================================================

Совет 1: Используйте compiler optimizations
-------------------------------------------
// МЕДЛЕННО (без оптимизаций):
gcc -o program program.c

// БЫСТРО (с оптимизациями):
gcc -O3 -o program program.c

// ⚡ -O3 ускоряет в 5-10x!

Совет 2: Используйте inline функции
-----------------------------------
// МЕДЛЕННО (function call overhead):
int square(int x) {
    return x * x;
}

// БЫСТРО (inline):
inline int square(int x) {
    return x * x;
}

// ⚡ Inline предотвращает function call overhead!

Совет 3: Используйте restrict keyword
-------------------------------------
// МЕДЛЕННО (compiler не знает):
void add(int* a, int* b, int* c, int n) {
    for (int i = 0; i < n; i++) {
        c[i] = a[i] + b[i];
    }
}

// БЫСТРО (restrict):
void add(int* restrict a, int* restrict b, int* restrict c, int n) {
    for (int i = 0; i < n; i++) {
        c[i] = a[i] + b[i];
    }
}

// ⚡ Restrict позволяет better optimizations!

Совет 4: Используйте loop unrolling
-----------------------------------
// МЕДЛЕННО (normal loop):
for (int i = 0; i < 100; i++) {
    sum += arr[i];
}

// БЫСТРО (unrolled):
for (int i = 0; i < 100; i += 4) {
    sum += arr[i];
    sum += arr[i+1];
    sum += arr[i+2];
    sum += arr[i+3];
}

// ⚡ Unrolling reduces loop overhead!

Совет 5: Используйте cache-friendly access
------------------------------------------
// МЕДЛЕННО (column-major):
for (int j = 0; j < N; j++) {
    for (int i = 0; i < N; i++) {
        sum += matrix[i][j];
    }
}

// БЫСТРО (row-major):
for (int i = 0; i < N; i++) {
    for (int j = 0; j < N; j++) {
        sum += matrix[i][j];
    }
}

// ⚡ Row-major лучше для cache!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.c:
---------------------
#include <stdio.h>
#include <time.h>

int main() {
    clock_t start = clock();
    
    long long sum = 0;
    for (long long i = 0; i < 1000000000; i++) {
        sum += i;
    }
    
    clock_t end = clock();
    double time = (double)(end - start) / CLOCKS_PER_SEC * 1000;
    
    printf("Sum: %lld\n", sum);
    printf("Time: %.2f ms\n", time);
    
    return 0;
}

Компиляция и запуск:
--------------------
gcc -O3 -o benchmark benchmark.c
time ./benchmark

Сравнение с Python:
-------------------
# C будет в 50-100 раз быстрее!
