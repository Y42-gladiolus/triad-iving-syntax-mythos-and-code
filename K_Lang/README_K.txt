5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: K5 требует лицензию
   K5 не запускается без лицензии
   
   Решение: 
   Запросить бесплатную лицензию на kx.com
   Или использовать Kona/oK

⚠️ Проблема 2: Kona не полностью совместим с K5
   Некоторые функции K5 отсутствуют в Kona
   
   Решение:
   Использовать Kona для обучения
   K5 для production

⚠️ Проблема 3: Синтаксис очень отличается
   K использует символы вместо слов
   
   Решение:
   Распечатать шпаргалку по символам
   Практиковаться ежедневно

⚠️ Проблема 4: Справа-налево вычисление
   2*3+4 = 14 в K (не 10!)
   
   Решение:
   Привыкнуть к right-to-left evaluation
   Использовать скобки для ясности

⚠️ Проблема 5: Мало документации
   Официальная документация платная
   
   Решение:
   Использовать wiki.kx.com
   Изучать примеры на GitHub

# Добавить в ~/.bashrc, ~/.zshrc, ~/.mkshrc

echo "alias k='kona'" >> ~/.zshrc

echo "alias k5='k5'" >> ~/.bashrc
echo "alias k5='k5'" >> ~/.zshrc

echo "alias q='q'" >> ~/.bashrc
echo "alias q='q'" >> ~/.zshrc

# Применить изменения
source ~/.bashrc
source ~/.zshrc
source ~/.mkshrc 2>/dev/null || true
