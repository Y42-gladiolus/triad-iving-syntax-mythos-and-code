3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ PHP
================================================================================

Совет 1: Используйте OPCache
----------------------------
# МЕДЛЕННО (без кэша):
php script.php

# БЫСТРО (с OPCache):
php -d opcache.enable=1 script.php

# ⚡ OPCache ускоряет в 5-10x!

Совет 2: Используйте PHP 8+ с JIT
---------------------------------
# МЕДЛЕННО (PHP 7.x):
php7 script.php

# БЫСТРО (PHP 8+ с JIT):
php8 -d opcache.jit_buffer_size=256M script.php

# ⚡ JIT ускоряет CPU-bound tasks!

Совет 3: Используйте prepared statements
----------------------------------------
# МЕДЛЕННО (SQL injection risk):
$query = "SELECT * FROM users WHERE id = " . $_GET['id'];

# БЫСТРО (prepared):
$stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([$_GET['id']]);

# ⚡ Prepared statements безопаснее и быстрее!

Совет 4: Используйте autoloading
--------------------------------
# МЕДЛЕННО (manual includes):
require 'file1.php';
require 'file2.php';

# БЫСТРО (Composer autoloading):
require 'vendor/autoload.php';

# ⚡ Autoloading эффективнее!

Совет 5: Используйте caching
----------------------------
# МЕДЛЕННО (database queries каждый раз):
$users = $db->query("SELECT * FROM users");

# БЫСТРО (с кэшем):
$users = $cache->remember('users', 3600, function() use ($db) {
    return $db->query("SELECT * FROM users");
});

# ⚡ Caching уменьшает database load!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.php:
-----------------------
<?php
function sumEnum($n) {
    $sum = 0;
    for ($i = 1; $i <= $n; $i++) {
        $sum += $i;
    }
    return $sum;
}

$start = microtime(true);
$result = sumEnum(100000000);
$end = microtime(true);

echo "Sum: $result\n";
echo "Time: " . (($end - $start) * 1000) . " ms\n";

Запуск:
-------
php benchmark.php
php -d opcache.enable=1 benchmark.php  # С OPCache
