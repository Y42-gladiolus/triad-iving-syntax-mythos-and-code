3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ HASKELL
================================================================================

Совет 1: Используйте strictness когда нужно
-------------------------------------------
-- МЕДЛЕННО (lazy):
sum :: [Int] -> Int
sum xs = foldl (+) 0 xs

-- БЫСТРО (strict):
sum :: [Int] -> Int
sum xs = foldl' (+) 0 xs

-- ⚡ Strict foldl предотвращает space leaks!

Совет 2: Используйте -O2 для оптимизаций
----------------------------------------
-- МЕДЛЕННО (без оптимизаций):
ghc Main.hs

-- БЫСТРО (с оптимизациями):
ghc -O2 Main.hs

-- ⚡ -O2 ускоряет в 5-10x!

Совет 3: Используйте Unboxed types
----------------------------------
-- МЕДЛЕННО (boxed):
sum :: [Int] -> Int

-- БЫСТРО (unboxed):
sum :: [Int#] -> Int#

-- ⚡ Unboxed types быстрее но менее portable!

Совет 4: Используйте Parallel strategies
----------------------------------------
-- МЕДЛЕННО (последовательно):
result = map expensive [1..1000]

-- БЫСТРО (параллельно):
import Control.Parallel.Strategies
result = map expensive [1..1000] `using` parList rseq

-- ⚡ Parallel быстрее на multi-core!

Совет 5: Используйте LLVM backend
---------------------------------
-- МЕДЛЕННО (native code):
ghc -O2 Main.hs

-- БЫСТРО (LLVM):
ghc -O2 -fllvm Main.hs

-- ⚡ LLVM может ускорить на 10-20%!

sumEnum :: Integer -> Integer
sumEnum n = sum [1..n]

main :: IO ()
main = do
  let result = sumEnum 100000000
  putStrLn $ "Sum: " ++ show result

--------------------
