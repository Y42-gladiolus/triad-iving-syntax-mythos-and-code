Linux (Ubuntu/Debian):
----------------------
# Вариант 1: Скачать бинарный дистрибутив
wget https://factorcode.org/static/factor-0.98-linux-x86-64.tar.gz
tar xzf factor-0.98-linux-x86-64.tar.gz
cd factor-0.98
./factor

# Вариант 2: Через пакетный менеджер (может быть старая версия)
sudo apt update
sudo apt install factor

# Добавить в PATH
export PATH="$HOME/factor-0.98:$PATH"
echo 'export PATH="$HOME/factor-0.98:$PATH"' >> ~/.bashrc

# Проверка
factor --version

macOS (Homebrew):
-----------------
brew install factor

# Проверка
factor --version

Windows:
--------
# Скачать с https://factorcode.org/
# Распаковать архив
# factor.exe в корне папки

# Проверка в PowerShell:
.\factor.exe --version

Docker (универсально):
----------------------
docker run -it --rm factorcode/factor factor

3. БЫСТРЫЙ СТАРТ
================================================================================

# Factor REPL
factor

# В REPL:
IN: scratchpad 2 2 + .
4
IN: scratchpad bye  # Выход

# Запуск скрипта
factor script.factor

# Запуск с кодом из командной строки
factor -eval="2 2 + ."

# Компиляция в образ
factor -image=script.image script.factor

# Создание проекта

5. ПОДВОДНЫЕ КАМНИ УСТАНОВКИ
================================================================================

⚠️ Проблема 1: PATH не настроен
   factor: command not found
   
   Решение:
   export PATH="$HOME/factor-0.98:$PATH"
   Добавить в ~/.bashrc

⚠️ Проблема 2: GUI библиотеки не найдены
   Factor IDE не запускается
   
   Решение:
   sudo apt install libgtk-3-dev libwebkit2gtk-4.0-dev

⚠️ Проблема 3: Stack-based нотация непривычна
   2 2 + вместо 2 + 2
   
   Решение:
   Практиковаться ежедневно
   Использовать встроенную документацию

⚠️ Проблема 4: Много памяти при запуске
   Factor требует ~50-100 MB RAM
   
   Решение:
   Это нормально для богатой стандартной библиотеки
   Использовать -no-ui для минимального режима

⚠️ Проблема 5: Компиляция образа долгая
   Первый запуск может занять время
   
   Решение:
   Это нормально - компиляция в образ
   Последующие запуски быстрые

⚠️ Проблема 6: Разные версии Factor
   Старые скрипты могут не работать
   
   Решение:
   Использовать последнюю стабильную версию
   Проверять совместимость
