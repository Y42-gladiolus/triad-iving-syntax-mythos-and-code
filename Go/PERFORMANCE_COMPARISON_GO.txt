3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ GO
================================================================================

Совет 1: Используйте goroutines для I/O-bound
---------------------------------------------
# МЕДЛЕННО (последовательно):
for _, url := range urls {
    resp, _ := http.Get(url)
    // Process
}

# БЫСТРО (параллельно):
for _, url := range urls {
    go func(u string) {
        resp, _ := http.Get(u)
        // Process
    }(url)
}

# ⚡ Goroutines ускоряют I/O-bound tasks!

Совет 2: Используйте sync.Pool для объектов
-------------------------------------------
# МЕДЛЕННО (allocation каждый раз):
for {
    buf := make([]byte, 1024)
    // Use buf
}

# БЫСТРО (reuse):
pool := sync.Pool{
    New: func() interface{} {
        return make([]byte, 1024)
    },
}
for {
    buf := pool.Get().([]byte)
    // Use buf
    pool.Put(buf)
}

# ⚡ sync.Pool уменьшает GC pressure!

Совет 3: Используйте buffered channels
--------------------------------------
# МЕДЛЕННО (unbuffered):
ch := make(chan int)

# БЫСТРО (buffered):
ch := make(chan int, 100)

# ⚡ Buffered channels уменьшают блокировки!

Совет 4: Используйте go build -ldflags
--------------------------------------
# МЕДЛЕННО (с debug info):
go build -o app main.go

# БЫСТРО (без debug info):
go build -ldflags="-s -w" -o app main.go

# ⚡ Уменьшает binary size на 20-30%!

Совет 5: Используйте pprof для profiling
----------------------------------------
# Профилирование:
import _ "net/http/pprof"
go func() {
    http.ListenAndServe("localhost:6060", nil)
}()

# Анализ:
go tool pprof http://localhost:6060/debug/pprof/heap

# ⚡ pprof помогает найти bottlenecks!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.go:
----------------------
package main

import (
    "fmt"
    "time"
)

func sumEnum(n int) int64 {
    var sum int64 = 0
    for i := 1; i <= n; i++ {
        sum += int64(i)
    }
    return sum
}

func main() {
    start := time.Now()
    result := sumEnum(1000000000)
    elapsed := time.Since(start)
    
    fmt.Println("Sum:", result)
    fmt.Println("Time:", elapsed.Milliseconds(), "ms")
}

Запуск:
-------
go run benchmark.go
go build -o benchmark benchmark.go && ./benchmark

Сравнение с Python:
-------------------
# Go будет в 50-100 раз быстрее Python!
