3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ CARBON
================================================================================

Совет 1: Используйте release mode
---------------------------------
# МЕДЛЕННО (debug):
bazel build //:main

# БЫСТРО (release):
bazel build -c opt //:main

# ⚡ Release mode ускоряет в 5-10x!

Совет 2: Используйте incremental build
--------------------------------------
# МЕДЛЕННО (full rebuild):
bazel build //:main

# БЫСТРО (incremental):
bazel build //:main --incremental

# ⚡ Incremental build быстрее!

Совет 3: Используйте C++ interop
--------------------------------
# МЕДЛЕННО (rewrite):
// Переписывать C++ код на Carbon

# БЫСТРО (interop):
import Cpp mylib;
Cpp.MyFunction();

# ⚡ C++ interop без overhead!

Совет 4: Используйте generics вместо templates
----------------------------------------------
# МЕДЛЕННО (C++ templates):
// template<typename T> T Max(T a, T b)

# БЫСТРО (Carbon generics):
fn Max[T: Comparable](a: T, b: T) -> T

# ⚡ Generics быстрее компилируются!

Совет 5: Используйте Bazel cache
--------------------------------
# МЕДЛЕННО (без cache):
bazel build //:main

# БЫСТРО (с cache):
bazel build //:main --disk_cache=/path/to/cache

# ⚡ Bazel cache ускоряет повторные сборки!

  var sum: i64 = 0;

  Print("Sum: " + sum);
  return 0;
