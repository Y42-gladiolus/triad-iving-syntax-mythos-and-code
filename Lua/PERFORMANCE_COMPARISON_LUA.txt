3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ LUA
================================================================================

Совет 1: Используйте LuaJIT
---------------------------
-- МЕДЛЕННО (стандартный Lua):
lua script.lua

-- БЫСТРО (LuaJIT):
luajit script.lua

-- ⚡ Ускорение: 10-100x для CPU-bound!

Совет 2: Используйте local переменные
-------------------------------------
-- МЕДЛЕННО (глобальные):
for i = 1, 1000000 do
    x = math.sqrt(i)  -- Глобальный lookup!
end

-- БЫСТРО (local):
local sqrt = math.sqrt
for i = 1, 1000000 do
    local x = sqrt(i)  -- Local lookup!
end

-- ⚡ Ускорение: 2-5x!

Совет 3: Избегайте таблиц в горячих циклах
------------------------------------------
-- МЕДЛЕННО (создание таблиц):
for i = 1, 1000000 do
    local t = {x = i, y = i * 2}
end

-- БЫСТРО (переиспользование):
local t = {}
for i = 1, 1000000 do
    t.x = i
    t.y = i * 2
end

-- ⚡ Меньше аллокаций = быстрее!

Совет 4: Используйте table.concat для строк
-------------------------------------------
-- МЕДЛЕННО (конкатенация):
local result = ""
for i = 1, 1000 do
    result = result .. i
end

-- БЫСТРО (table.concat):
local parts = {}
for i = 1, 1000 do
    parts[i] = tostring(i)
end
local result = table.concat(parts)

-- ⚡ Ускорение: 100-1000x для больших строк!

Совет 5: Избегайте метатаблиц в горячих путях
---------------------------------------------
-- МЕДЛЕННО (метатаблицы):
local t = setmetatable({}, mt)

-- БЫСТРО (простые таблицы):
local t = {}

-- ⚡ Метатаблицы имеют оверхед!

8. ИНТЕГРАЦИЯ LUA С ДРУГИМИ ЯЗЫКАМИ

Lua + C (нативная):
-------------------
// C код вызывает Lua
lua_State *L = luaL_newstate();
luaL_openlibs(L);
luaL_dostring(L, "print('Hello from Lua!')");
lua_close(L);

Lua + C++ (через sol2):
-----------------------
// C++ с sol2 библиотекой
sol::state lua;
lua.script("result = 2 + 2");
int result = lua["result"];

Lua + Python (через lupa):
--------------------------
import lupa
lua = lupa.LuaRuntime()
result = lua.eval("2 + 2")

Lua + MiniZinc (через CLI):
---------------------------
local handle = io.popen("minizinc --solver CBC model.mzn data.dzn")
local result = handle:read("*a")
handle:close()

Lua + Redis (скрипты):
----------------------
-- Redis Lua скрипт
local keys = KEYS[1]
local values = ARGV[1]
redis.call('SET', keys, values)

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте benchmark.lua:
-----------------------
local function sum_enum(n)
    local sum = 0
    for i = 1, n do
        sum = sum + i
    end
    return sum
end

local start = os.clock()
local result = sum_enum(100_000_000)
local finish = os.clock()

print("Sum:", result)
print("Time:", finish - start, "seconds")

Запуск:
-------
lua benchmark.lua

Сравнение с LuaJIT:
-------------------
luajit benchmark.lua

-- LuaJIT будет в 10-50 раз быстрее!
