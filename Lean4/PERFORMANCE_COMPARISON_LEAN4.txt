Совет 1: Используйте кэш Olean
------------------------------
# Первый build (медленно):
lake build

# Последующие (быстро, использует кэш):
lake build

-- Кэш хранится в ~/.cache/lean

Совет 2: Избегайте доказательств в runtime коде
-----------------------------------------------
-- МЕДЛЕННО (доказательство в runtime):
def process (xs : List α) (h : xs.length > 0) : α :=
  head xs h

-- БЫСТРО (доказательство в типе):
def process (xs : List α) (h : xs ≠ []) : α :=
  head xs h

-- ⚡ Доказательство проверяется на compile-time!

Совет 3: Используйте simp с нужными леммами
-------------------------------------------
-- МЕДЛЕННО (simp ищет все леммы):
theorem slow : n + 0 = n := by
  simp

-- БЫСТРО (конкретные леммы):
theorem fast : n + 0 = n := by
  simp [Nat.add_zero]

-- ⚡ Ускорение: 2-5x для сложных доказательств!

Совет 4: Используйте omega для арифметики
-----------------------------------------
-- МЕДЛЕННО (ручное доказательство):
theorem slow_omega (a b : Nat) : a < b → a + 1 ≤ b := by
  intro h
  -- Много шагов...

-- БЫСТРО (автоматически):
theorem fast_omega (a b : Nat) : a < b → a + 1 ≤ b := by
  intro h
  omega

-- ⚡ Ускорение: 10-100x для арифметики!

Совет 5: Используйте ring для колец
-----------------------------------
-- МЕДЛЕННО (ручное доказательство):
theorem slow_ring (a b c : Nat) : 
  (a + b) * c = a * c + b * c := by
  -- Много шагов...

-- БЫСТРО (автоматически):
theorem fast_ring (a b c : Nat) : 
  (a + b) * c = a * c + b * c := by
  ring

-- ⚡ Ускорение: 10-100x для алгебры!

9. БЕНЧМАРК: ПРОВЕРЬТЕ САМИ
================================================================================

Создайте Benchmark.lean:
------------------------
import Mathlib

def sumTo (n : Nat) : Nat :=
  if h : n = 0 then 0
  else n + sumTo (n - 1)

theorem sumTo_formula (n : Nat) : 
  sumTo n = n * (n + 1) / 2 := by
  induction n with
  | zero => simp [sumTo]
  | succ n ih => 
    simp [sumTo, ih]
    ring

#eval sumTo 1000

main : IO Unit := do
  let start ← IO.monoMsNow
  let result := sumTo 1000000
  let end ← IO.monoMsNow
  IO.println s!"Sum: {result}"
  IO.println s!"Time: {end - start}ms"

Компиляция и запуск:
--------------------
lake build
lake run benchmark

Сравнение с доказательствами vs без:
------------------------------------
-- С доказательствами (медленнее компиляция, гарантии)
theorem correct : sumTo n = n * (n + 1) / 2 := by
  induction n <;> simp [sumTo] <;> ring

-- Без доказательств (быстрее компиляция, нет гарантий)
def sumTo_unsafe (n : Nat) : Nat :=
  if n = 0 then 0 else n + sumTo_unsafe (n - 1)
