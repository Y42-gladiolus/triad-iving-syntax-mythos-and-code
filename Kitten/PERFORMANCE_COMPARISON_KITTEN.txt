3. ОПТИМИЗАЦИЯ: КАК УСКОРИТЬ KITTEN
================================================================================

Совет 1: Используйте type inference
-----------------------------------
\ МЕДЛЕННО (избыточные аннотации):
add (a: Int, b: Int) -> Int is
  a: Int b: Int +: Int

\ БЫСТРО (inference):
add (a, b) is
  a b +

\ ⚡ Компилятор выводит типы автоматически!

Совет 2: Избегайте mutable state
--------------------------------
\ МЕДЛЕННО (mutable):
var count = 0
count := count + 1

\ БЫСТРО (immutable):
let count = 0
let new_count = count + 1

\ ⚡ Immutable оптимизируется лучше!

Совет 3: Используйте tail-рекурсию
----------------------------------
\ МЕДЛЕННО (обычная рекурсия):
factorial (n) is
  n 1 <= if
    1
  else
    n n 1 - factorial *

\ БЫСТРО (tail-рекурсия):
factorial (n, acc = 1) is
  n 1 <= if
    acc
  else
    n 1 - acc n * factorial

\ ⚡ Kitten оптимизирует tail-рекурсию!

Совет 4: Компилируйте в release mode
------------------------------------
\ МЕДЛЕННО (debug):
kitten script.ktn

\ БЫСТРО (release):
kitten --release script.ktn

\ ⚡ Ускорение: 5-10x для runtime!

Совет 5: Используйте встроенные функции
---------------------------------------
\ МЕДЛЕННО (ручная реализация):
sum (list) is
  list 0 fold(+)

\ БЫСТРО (встроенная):
list sum

\ ⚡ Встроенные функции оптимизированы!
